Menu contextuel

Un clic droit dans l'éditeur ouvre un menu contextuel avec les actions d'édition courantes (Annuler, Couper, Copier, Coller, Tout sélectionner, Rechercher, Remplacer, Reformater le code).

Les raccourcis utilisant Ctrl fonctionnent aussi avec Cmd sur macOS.

À propos

Visual Objects Handler

Framework JavaScript pour la création d'éditeurs visuels interactifs.

Manipulation d'objets graphiques sur canvas avec API complète.

Concepteur: David Dieppedalle (ShadowStorm)

Origine: Portage web du concept Editors Factory (PureBasic)

Bibliothèques utilisées:

Pixi.js (MIT) · CodeMirror (MIT) · TypeScript (Apache 2.0)
