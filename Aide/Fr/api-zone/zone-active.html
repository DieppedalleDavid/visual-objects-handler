voh.zone — Zone active

Gérer quelle zone est visible à l'écran.

Une seule zone est visible à la fois. Quand on active une zone, les autres sont masquées. Chaque zone garde ses propriétés en mémoire même quand elle n'est pas visible.
voh.zone.setActive(zoneId) → vrai/faux
Active une zone (la rend visible) et masque toutes les autres.
Retourne: vrai/faux — true si réussi, false si zone inexistante.
const zone1 = voh.zone.create({ name: 'Rouge', backgroundColor: 'rgba(180, 40, 40, 1.0)' });
const zone2 = voh.zone.create({ name: 'Bleue', backgroundColor: 'rgba(40, 40, 180, 1.0)' });

// Afficher la zone bleue
voh.zone.setActive(zone2);

// Revenir sur la rouge
voh.zone.setActive(zone1);
voh.zone.getActiveId() → nombre ou null
Retourne l'ID de la zone actuellement visible. Retourne null si aucune zone n'existe.
Retourne: nombre — l'ID de la zone active, ou null si aucune zone n'existe.
const active = voh.zone.getActiveId();
console.log('Zone visible:', active);  // ex: 1
