voh.zone — Informations

Obtenir des informations sur les zones existantes.

voh.zone.getCount() → nombre
Retourne combien de zones existent au total.
Retourne: nombre — le nombre de zones (0 si aucune).
console.log(voh.zone.getCount());  // ex: 3
voh.zone.getList() → tableau de nombres
Retourne la liste des IDs de toutes les zones.
Retourne: tableau — les IDs des zones (tableau vide [] si aucune).
console.log(voh.zone.getList());  // ex: [1, 2, 3]
voh.zone.exists(zoneId) → vrai/faux
Vérifie si une zone existe encore (utile après une suppression).
Retourne: vrai/faux — true si la zone existe, false sinon.
console.log(voh.zone.exists(1));  // true ou false

Conteneur de canevas

voh.zone.getCanvasContainer(zoneId) → HTMLElement | null
Retourne l'élément DOM conteneur des canevas à l'intérieur de la zone. C'est le div scrollable dans lequel les canevas sont positionnés. Retourne null si la zone n'existe pas.
Retourne: HTMLElement | null
const container = voh.zone.getCanvasContainer(zone);
console.log(container);  // <div class="voh-zone-canvas-container">

Auto-front à l'activation

voh.zone.setBringCanvasToFrontOnActivation(zoneId, enabled) → vrai/faux
Active ou désactive le comportement qui monte automatiquement un canevas au premier plan quand il est activé. Par défaut: false (le z-order ne change pas à l'activation).
ParamètreTypeDescription
zoneIdnombreL'ID de la zone
enabledvrai/fauxtrue = auto-front activé, false = désactivé
Retourne: vrai/faux
voh.zone.setBringCanvasToFrontOnActivation(zone, true);
voh.zone.getBringCanvasToFrontOnActivation(zoneId) → vrai/faux | null
Retourne true si l'auto-front est activé, false sinon, ou null si la zone n'existe pas.
Retourne: vrai/faux | null
voh.zone.getBringCanvasToFrontOnActivation(zone);  // true ou false
