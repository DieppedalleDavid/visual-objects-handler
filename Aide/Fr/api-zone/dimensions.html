voh.zone — Dimensions

Définir et lire la taille des zones.

Par défaut, width et height valent null: la zone prend 100% de l'espace disponible. Si on définit une taille en pixels plus grande que le conteneur, des scrollbars apparaissent.
voh.zone.setWidth(zoneId, largeur) / voh.zone.getWidth(zoneId)
Définit ou lit la largeur. Passer null pour reprendre 100%.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setWidth(zone, 1200);       // Largeur fixe 1200px
voh.zone.setWidth(zone, null);       // Reprendre 100%
console.log(voh.zone.getWidth(zone));  // null ou 1200
voh.zone.setHeight(zoneId, hauteur) / voh.zone.getHeight(zoneId)
Définit ou lit la hauteur. Même logique que la largeur.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setHeight(zone, 800);
console.log(voh.zone.getHeight(zone));  // 800
