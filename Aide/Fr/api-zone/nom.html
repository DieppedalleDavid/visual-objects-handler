voh.zone — Nom

Attribuer un nom à une zone pour l'identifier.

voh.zone.setName(zoneId, nom) / voh.zone.getName(zoneId)
Définit ou lit le nom de la zone. Le nom sert à l'identifier, il n'est pas affiché visuellement.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setName(zone, 'Éditeur principal');
console.log(voh.zone.getName(zone));  // 'Éditeur principal'
