voh.zone — Zoom

Contrôler le niveau de zoom de la zone.

Le zoom va de 10% (0.1) à 500% (5.0). 1.0 = taille normale (100%). 2.0 = deux fois plus gros. 0.5 = deux fois plus petit.
voh.zone.setZoom(zoneId, niveau) / voh.zone.getZoom(zoneId)
Définit ou lit le niveau de zoom.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.zoomIn(zoneId) → vrai/faux
Augmente le zoom de 10% (un palier).
Retourne: vrai/faux — true si réussi, false si zone inexistante.
voh.zone.zoomOut(zoneId) → vrai/faux
Diminue le zoom de 10% (un palier).
Retourne: vrai/faux — true si réussi, false si zone inexistante.
voh.zone.resetZoom(zoneId) → vrai/faux
Remet le zoom à 100%.
Retourne: vrai/faux — true si réussi, false si zone inexistante.
// Zoom à 150%
voh.zone.setZoom(zone, 1.5);

// Zoomer par palier (+10% à chaque appel)
voh.zone.zoomIn(zone);   // → 160%
voh.zone.zoomIn(zone);   // → 170%

// Dézoomer par palier (-10% à chaque appel)
voh.zone.zoomOut(zone);  // → 160%

// Lire le zoom actuel
console.log(voh.zone.getZoom(zone));  // 1.6

// Revenir à 100%
voh.zone.resetZoom(zone);
