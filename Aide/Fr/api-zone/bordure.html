voh.zone — Bordure

Couleur et épaisseur de la bordure autour de la zone.

voh.zone.setBorderColor(zoneId, couleur) / voh.zone.getBorderColor(zoneId)
Couleur de la bordure autour de la zone.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setBorderThickness(zoneId, pixels) / voh.zone.getBorderThickness(zoneId)
Épaisseur de la bordure en pixels. 0 = pas de bordure visible.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
// Bordure rouge épaisse
voh.zone.setBorderColor(zone, 'rgba(255, 0, 0, 1.0)');
voh.zone.setBorderThickness(zone, 3);

// Supprimer la bordure
voh.zone.setBorderThickness(zone, 0);
