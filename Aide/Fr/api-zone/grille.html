voh.zone — Grille

Quadrillage visuel sur la zone pour positionner les éléments.

La grille est un quadrillage visuel sur la zone qui sert de repère pour positionner les canevas. Elle est masquée par défaut — il faut passer gridIsVisible: true pour la voir.
voh.zone.setGridWidth(zoneId, pixels) / voh.zone.getGridWidth(zoneId)
Espacement horizontal entre les lignes de la grille (en pixels, minimum 1).
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setGridHeight(zoneId, pixels) / voh.zone.getGridHeight(zoneId)
Espacement vertical entre les lignes de la grille (en pixels, minimum 1).
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setGridColor(zoneId, couleur) / voh.zone.getGridColor(zoneId)
Couleur des lignes de la grille.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setGridOpacity(zoneId, opacité) / voh.zone.getGridOpacity(zoneId)
Opacité de la grille. 0.0 = invisible, 1.0 = opaque.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setGridVisible(zoneId, visible) / voh.zone.getGridVisible(zoneId)
Affiche ou masque la grille. Les propriétés sont conservées même quand la grille est masquée.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
// Activer la grille et la configurer
voh.zone.setGridVisible(zone, true);
voh.zone.setGridWidth(zone, 40);
voh.zone.setGridHeight(zone, 40);
voh.zone.setGridColor(zone, 'rgba(255, 255, 255, 1.0)');
voh.zone.setGridOpacity(zone, 0.2);

// Masquer la grille (les réglages sont conservés)
voh.zone.setGridVisible(zone, false);

// La réafficher plus tard → elle reprend son apparence
voh.zone.setGridVisible(zone, true);
