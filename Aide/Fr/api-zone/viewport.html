voh.zone — Viewport (Scroll)

Contrôler le défilement et les scrollbars de la zone.

Le viewport est la fenêtre visible de la zone. Quand la zone est plus grande que le conteneur (par exemple 3000×2000 pixels), on ne voit qu'une partie à la fois. Les scrollbars permettent de naviguer, et on peut aussi contrôler le défilement par code.
voh.zone.setScrollbarVisible(zoneId, visible) / voh.zone.getScrollbarVisible(zoneId)
true = scrollbars automatiques, false = masquées. Même sans scrollbars, le scroll par code fonctionne toujours.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setScrollX(zoneId, x) / voh.zone.getScrollX(zoneId)
Position horizontale du défilement en pixels. 0 = bord gauche.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setScrollY(zoneId, y) / voh.zone.getScrollY(zoneId)
Position verticale du défilement en pixels. 0 = bord haut.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.scrollToCenter(zoneId) → vrai/faux
Centre la vue pour que le milieu de la zone soit au milieu de l'écran.
Retourne: vrai/faux — true si réussi, false si zone inexistante.
// Défiler vers le coin haut-gauche
voh.zone.setScrollX(zone, 0);
voh.zone.setScrollY(zone, 0);

// Défiler vers une position précise
voh.zone.setScrollX(zone, 500);
voh.zone.setScrollY(zone, 300);

// Lire la position actuelle
console.log('Position:', voh.zone.getScrollX(zone), voh.zone.getScrollY(zone));

// Centrer la vue
voh.zone.scrollToCenter(zone);
