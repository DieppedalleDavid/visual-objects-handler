voh.zone — Création / Suppression

Créer de nouvelles zones et les supprimer.

Créer une zone

voh.zone.create(options) → nombre
Crée une nouvelle zone et retourne son ID (un nombre unique). Cet ID est ensuite utilisé dans toutes les autres fonctions pour désigner cette zone. Tous les paramètres sont optionnels — si on ne précise rien, la zone utilise les valeurs par défaut.
ParamètreTypeDescription
nametexteNom de la zone (pour l'identifier)
widthnombre ou nullLargeur en pixels. null = prend toute la place
heightnombre ou nullHauteur en pixels. null = prend toute la place
backgroundColorcouleurCouleur de fond (ex: 'rgba(45, 45, 45, 1.0)')
backgroundImagetexteImage de fond (URL ou data:URI)
backgroundImageOpacitynombreOpacité de l'image: 0.0 = invisible → 1.0 = opaque
borderColorcouleurCouleur de la bordure
borderThicknessnombreÉpaisseur de la bordure en pixels (0 = pas de bordure)
gridWidthnombreEspacement horizontal de la grille en pixels
gridHeightnombreEspacement vertical de la grille en pixels
gridColorcouleurCouleur des lignes de la grille
gridOpacitynombreOpacité de la grille: 0.0 → 1.0
gridIsVisiblevrai/fauxtrue = grille affichée, false = masquée
scrollbarIsVisiblevrai/fauxtrue = scrollbars visibles
zoomnombreNiveau de zoom: 1.0 = 100%
Retourne: nombre — L'ID de la zone créée (nombre ≥ 1). La création réussit toujours.
// Zone simple avec fond bleu
const zone = voh.zone.create({
    name: 'Ma zone',
    backgroundColor: 'rgba(40, 80, 120, 1.0)'
});

// Zone complète avec grille, taille fixe et bordure
const zone2 = voh.zone.create({
    name: 'Éditeur principal',
    width: 2000,
    height: 1500,
    backgroundColor: 'rgba(30, 30, 30, 1.0)',
    borderColor: 'rgba(255, 0, 0, 1.0)',
    borderThickness: 2,
    gridWidth: 25,
    gridHeight: 25,
    gridColor: 'rgba(255, 255, 255, 1.0)',
    gridOpacity: 0.15,
    gridIsVisible: true
});

Supprimer une zone

voh.zone.delete(zoneId, options) → vrai/faux
Supprime une zone. Si c'était la zone active (celle affichée à l'écran), VOH bascule automatiquement sur une autre zone. Si c'était la dernière, il n'y a plus rien d'affiché. Par défaut, les canvas contenus dans la zone sont aussi supprimés.
ParamètreTypeDescription
zoneIdnombreL'ID de la zone à supprimer
options.deleteCanvasesvrai/fauxtrue = supprime aussi les canvas de la zone (par défaut: true). false = les canvas restent en mémoire (utile pour migration)
Retourne: vrai/faux — true si la zone a été supprimée, false si la zone n'existe pas.
// Suppression standard (les canvas sont aussi supprimés)
voh.zone.delete(zone);

// Garder les canvas en mémoire pour les migrer
voh.zone.delete(zone, { deleteCanvases: false });
