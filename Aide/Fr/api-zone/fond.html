voh.zone — Fond

Couleur de fond, image de fond et opacité de l'image.

voh.zone.setBackgroundColor(zoneId, couleur) / voh.zone.getBackgroundColor(zoneId)
Couleur de fond. Accepte tout format CSS: rgba, hex, nom de couleur.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setBackgroundColor(zone, 'rgba(20, 40, 80, 1.0)');
voh.zone.setBackgroundColor(zone, '#2d2d2d');
console.log(voh.zone.getBackgroundColor(zone));
voh.zone.setBackgroundImage(zoneId, image) / voh.zone.getBackgroundImage(zoneId)
Image de fond. Accepte une URL ou un data:URI (base64). Passer une chaîne vide '' pour enlever l'image.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setBackgroundImage(zone, 'Medias/Images/fond.png');   // Depuis un fichier
voh.zone.setBackgroundImage(zone, '');                     // Enlever l'image
voh.zone.setBackgroundImageOpacity(zoneId, opacité) / voh.zone.getBackgroundImageOpacity(zoneId)
Opacité de l'image de fond. 0.0 = invisible, 0.5 = semi-transparent, 1.0 = opaque.
set → true si réussi, false si la zone n'existe pas. get → la valeur, ou null si la zone n'existe pas
voh.zone.setBackgroundImageOpacity(zone, 0.5);  // 50% de transparence
