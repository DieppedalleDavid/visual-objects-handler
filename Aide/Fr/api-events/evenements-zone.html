Événements — Zone

Les 12 types d'événements émis par les zones. Chaque modification émet un événement capté par voh.on().

Liste des événements

ÉvénementDonnées reçuesQuand
zone:createdzoneId, nameUne zone est créée
zone:deletedzoneId, nameUne zone est supprimée
zone:activatedzoneId, previousZoneIdUne zone devient active
zone:resizedzoneId, width, heightLa largeur ou hauteur change
zone:renamedzoneId, nameLe nom change
zone:backgroundChangedzoneId, property, valueLe fond change (couleur, image, opacité image)
zone:gridChangedzoneId, property, valueLa grille change (taille, couleur, opacité, visibilité)
zone:zoomChangedzoneId, zoom, previousZoomLe niveau de zoom change
zone:scrollChangedzoneId, scrollX, scrollYLa position de scroll change
zone:borderChangedzoneId, property, valueLa bordure change (couleur, épaisseur)
zone:scrollbarVisibleChangedzoneId, visibleLa visibilité des scrollbars change
zone:bringCanvasToFrontOnActivationChangedzoneId, enabledL'option de mise au premier plan auto change
22 émissions au total — chaque appel de setter émet un seul événement. Les événements ne sont émis que si la valeur change réellement (pas de doublon).

zone:created

Émis quand une nouvelle zone est créée avec voh.zone.create().
PropriétéTypeDescription
zoneIdnombreL'ID de la zone créée
nametexteLe nom de la zone
voh.on('zone:created', (data) => {
    console.log(data.zoneId);  // 1
    console.log(data.name);    // "Ma zone"
});

zone:deleted

Émis quand une zone est supprimée avec voh.zone.delete(). Le nom est capturé avant la suppression pour être disponible dans l'événement.
PropriétéTypeDescription
zoneIdnombreL'ID de la zone supprimée
nametexteLe nom de la zone (avant suppression)
voh.on('zone:deleted', (data) => {
    console.log('Zone supprimée:', data.name);
});

zone:activated

Émis quand on change de zone active avec voh.zone.setActive(). Également émis automatiquement lors de la création de la première zone ou après la suppression de la zone active.
PropriétéTypeDescription
zoneIdnombreL'ID de la nouvelle zone active
previousZoneIdnombre ou nullL'ID de la zone précédemment active. null si c'est la première activation
voh.on('zone:activated', (data) => {
    console.log('Active:', data.zoneId);
    console.log('Précédente:', data.previousZoneId);
});

zone:resized

Émis quand la taille d'une zone change avec voh.zone.setWidth() ou voh.zone.setHeight(). Les deux dimensions sont toujours fournies dans l'événement.
PropriétéTypeDescription
zoneIdnombreL'ID de la zone redimensionnée
widthnombre ou nullLargeur en pixels. null = 100%
heightnombre ou nullHauteur en pixels. null = 100%
voh.on('zone:resized', (data) => {
    console.log('Taille:', data.width, '×', data.height);
});

zone:renamed

Émis quand le nom d'une zone est modifié avec voh.zone.setName().
PropriétéTypeDescription
zoneIdnombreL'ID de la zone renommée
nametexteLe nouveau nom
voh.on('zone:renamed', (data) => {
    console.log('Nouveau nom:', data.name);
});

zone:backgroundChanged

Émis quand une propriété du fond change: voh.zone.setBackgroundColor(), voh.zone.setBackgroundImage() ou voh.zone.setBackgroundImageOpacity().
PropriétéTypeDescription
zoneIdnombreL'ID de la zone modifiée
propertytexteNom de la propriété modifiée (voir ci-dessous)
valuevariableLa nouvelle valeur

Valeurs possibles de property

propertyDéclencheurType de value
backgroundColorsetBackgroundColor()couleur
backgroundImagesetBackgroundImage()texte (URL)
backgroundImageOpacitysetBackgroundImageOpacity()nombre (0.0 → 1.0)
voh.on('zone:backgroundChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "backgroundColor" = "rgba(30, 60, 100, 1.0)"
});

zone:gridChanged

Émis quand une propriété de la grille change: setGridWidth(), setGridHeight(), setGridColor(), setGridOpacity() ou setGridVisible().
PropriétéTypeDescription
zoneIdnombreL'ID de la zone modifiée
propertytexteNom de la propriété modifiée (voir ci-dessous)
valuevariableLa nouvelle valeur

Valeurs possibles de property

propertyDéclencheurType de value
gridWidthsetGridWidth()nombre (pixels)
gridHeightsetGridHeight()nombre (pixels)
gridColorsetGridColor()couleur
gridOpacitysetGridOpacity()nombre (0.0 → 1.0)
gridIsVisiblesetGridVisible()vrai/faux
voh.on('zone:gridChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "gridWidth" = 50
    // "gridIsVisible" = true
});

zone:zoomChanged

Émis quand le niveau de zoom change avec voh.zone.setZoom(). Également déclenché par zoomIn(), zoomOut() et resetZoom() qui appellent setZoom() en interne.
PropriétéTypeDescription
zoneIdnombreL'ID de la zone
zoomnombreLe nouveau niveau de zoom (1.0 = 100%)
previousZoomnombreLe niveau de zoom précédent
voh.on('zone:zoomChanged', (data) => {
    console.log('Zoom:', data.zoom);          // 1.5
    console.log('Avant:', data.previousZoom);  // 1
});

zone:scrollChanged

Émis quand la position de scroll change avec voh.zone.setScrollX(), voh.zone.setScrollY() ou voh.zone.scrollToCenter(). Les deux coordonnées sont toujours fournies.
PropriétéTypeDescription
zoneIdnombreL'ID de la zone
scrollXnombrePosition horizontale du scroll en pixels
scrollYnombrePosition verticale du scroll en pixels
voh.on('zone:scrollChanged', (data) => {
    console.log('Scroll:', data.scrollX, data.scrollY);
});

zone:borderChanged

Émis quand une propriété de la bordure change: voh.zone.setBorderColor() ou voh.zone.setBorderThickness().
PropriétéTypeDescription
zoneIdnombreL'ID de la zone modifiée
propertytexte'borderColor' ou 'borderThickness'
valuevariableLa nouvelle valeur
voh.on('zone:borderChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "borderColor" = "rgba(255, 200, 0, 1.0)"
    // "borderThickness" = 4
});

zone:scrollbarVisibleChanged

Émis quand la visibilité des scrollbars change avec voh.zone.setScrollbarVisible().
PropriétéTypeDescription
zoneIdnombreL'ID de la zone modifiée
visiblevrai/fauxtrue = scrollbars visibles
voh.on('zone:scrollbarVisibleChanged', (data) => {
    console.log('Scrollbars:', data.visible);
});

zone:bringCanvasToFrontOnActivationChanged

Émis quand l'option de mise au premier plan automatique change avec voh.zone.setBringCanvasToFrontOnActivation().

Quand cette option est activée, activer un canvas dans la zone le place automatiquement devant tous les autres (z-order maximum), comme le comportement des fenêtres de bureau. Quand elle est désactivée (par défaut), activer un canvas ne modifie pas les z-orders.
PropriétéTypeDescription
zoneIdnombreL'ID de la zone modifiée
enabledvrai/fauxtrue = mise au premier plan automatique activée
voh.on('zone:bringCanvasToFrontOnActivationChanged', (data) => {
    console.log('Auto premier plan:', data.enabled);
});
