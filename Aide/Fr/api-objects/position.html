voh.objects — Position

Déplacer un objet sur le canevas (coordonnées X, Y).

Position X

voh.objects.setX(objectId, x) → vrai/faux
Définit la position horizontale de l'objet sur le canevas. (0, 0) est le coin haut-gauche du canevas. Les valeurs négatives sont autorisées (l'objet sort par le bord gauche).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
xnombrePosition X en pixels
Retourne: vrai/faux
voh.objects.getX(objectId) → nombre | null
Retourne la position X de l'objet.
Retourne: nombre | null

Position Y

voh.objects.setY(objectId, y) → vrai/faux
Définit la position verticale de l'objet sur le canevas.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
ynombrePosition Y en pixels
Retourne: vrai/faux
voh.objects.getY(objectId) → nombre | null
Retourne la position Y de l'objet.
Retourne: nombre | null
voh.objects.setX(objet, 200);
voh.objects.setY(objet, 150);
console.log(voh.objects.getX(objet), voh.objects.getY(objet));  // 200 150
