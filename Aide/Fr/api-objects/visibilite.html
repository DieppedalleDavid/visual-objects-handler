voh.objects — Visibilité

Masquer ou réafficher un objet (tout ou rien).

Masquer / Réafficher

voh.objects.setVisible(objectId, visible) → vrai/faux
Rend l'objet visible (true) ou le masque (false). Un objet masqué est retiré de l'affichage (display: none) mais reste en mémoire. Pour une transparence progressive, voir l'opacité.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
visiblevrai/fauxtrue = visible, false = masqué
Retourne: vrai/faux
voh.objects.setVisible(objet, false);  // masque
voh.objects.setVisible(objet, true);   // réaffiche

Lire l'état

voh.objects.isVisible(objectId) → vrai/faux | null
Retourne true si l'objet est visible, false s'il est masqué, ou null si l'objet n'existe pas.
Retourne: vrai/faux | null
if (voh.objects.isVisible(objet)) {
    console.log('L\'objet est visible.');
}
voh.objects.getVisible(objectId) → vrai/faux | null
Alias de isVisible(). Retourne la même valeur. Utile pour la cohérence avec les autres getters (getName, getWidth, etc.).
Retourne: vrai/faux | null
voh.objects.getVisible(objet);  // identique à isVisible()
