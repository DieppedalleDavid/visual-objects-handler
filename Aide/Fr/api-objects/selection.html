voh.objects.selection — Sélection d'objets

La sélection d'objets est gérée par canevas. Chaque canevas maintient sa propre liste d'objets sélectionnés. La sélection est vidée automatiquement au changement de page active.

Comportement interactif (automatique)

Quand un canevas a des objets avec mouseDragEnabled: true, la sélection interactive est automatique:

  • Clic gauche sur un objet → sélection exclusive (désélectionne les autres)
  • Ctrl+clic sur un objet → bascule (ajouter/retirer de la sélection)
  • Clic gauche sur le fond → désélectionner tous les objets
  • Rectangle de sélection → sélectionne tous les objets intersectés
  • Ctrl + rectangle → ajoute les objets intersectés à la sélection existante

Sélectionner / Désélectionner

voh.objects.selection.select(canvasId, objectId) → vrai/faux
Ajoute un objet à la sélection. Si l'objet est déjà sélectionné, ne fait rien. L'objet doit être visible et sur la page active.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
objectIdnombreL'ID de l'objet à sélectionner
Retourne: vrai/faux — true si l'objet a été ajouté
voh.objects.selection.select(canvas, objet);
voh.objects.selection.deselect(canvasId, objectId) → vrai/faux
Retire un objet de la sélection. Si l'objet n'est pas sélectionné, ne fait rien.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
objectIdnombreL'ID de l'objet à désélectionner
Retourne: vrai/faux — true si l'objet a été retiré
voh.objects.selection.deselect(canvas, objet);
voh.objects.selection.toggle(canvasId, objectId) → vrai/faux
Bascule la sélection d'un objet : le sélectionne s'il ne l'est pas, le désélectionne s'il l'est. C'est le comportement du Ctrl+clic.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
objectIdnombreL'ID de l'objet
Retourne: vrai/faux — true si l'état a changé
voh.objects.selection.toggle(canvas, objet);
voh.objects.selection.selectOnly(canvasId, objectId) → vrai/faux
Sélectionne un seul objet en désélectionnant tous les autres. C'est le comportement du clic simple (sans Ctrl).
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
objectIdnombreL'ID de l'objet à sélectionner
Retourne: vrai/faux — true si la sélection a changé
voh.objects.selection.selectOnly(canvas, objet3);
// Seul objet3 est maintenant sélectionné

Tout sélectionner / Tout désélectionner

voh.objects.selection.selectAll(canvasId) → vrai/faux
Sélectionne tous les objets visibles de la page active du canevas.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
Retourne: vrai/faux — true si au moins un objet a été ajouté
voh.objects.selection.selectAll(canvas);
voh.objects.selection.deselectAll(canvasId) → vrai/faux
Désélectionne tous les objets du canevas. Émet selection:cleared en plus de selection:changed.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
Retourne: vrai/faux — true si des objets ont été désélectionnés
voh.objects.selection.deselectAll(canvas);

Interroger la sélection

voh.objects.selection.getSelectedIds(canvasId) → nombre[]
Retourne un tableau contenant les IDs de tous les objets sélectionnés du canevas. Tableau vide si aucun.
Retourne: nombre[]
const ids = voh.objects.selection.getSelectedIds(canvas);
console.log('Sélectionnés:', ids);  // ex: [2, 5, 8]
voh.objects.selection.getSelectedCount(canvasId) → nombre
Retourne le nombre d'objets sélectionnés dans le canevas.
Retourne: nombre
const n = voh.objects.selection.getSelectedCount(canvas);
console.log(n + ' objet(s) sélectionné(s)');
voh.objects.selection.isSelected(canvasId, objectId) → vrai/faux
Vérifie si un objet est sélectionné dans un canevas donné.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
objectIdnombreL'ID de l'objet
Retourne: vrai/faux
if (voh.objects.selection.isSelected(canvas, objet)) {
    console.log('L\'objet est sélectionné');
}

Raccourci

voh.objects.isSelected(objectId) → vrai/faux
Raccourci qui retrouve automatiquement le canevas de l'objet. Pratique quand on a l'ID de l'objet mais pas celui du canevas.
Retourne: vrai/faux
if (voh.objects.isSelected(objet)) {
    console.log('Sélectionné !');
}

Événements

selection:changed
Émis chaque fois que la composition de la sélection change (ajout, retrait, remplacement). 
voh.on('selection:changed', function(data) {
    console.log('Canvas:',     data.canvasId);
    console.log('Sélectionnés:', data.selectedIds);   // tous les IDs après le changement
    console.log('Ajoutés:',      data.addedIds);      // IDs ajoutés dans cette action
    console.log('Retirés:',      data.removedIds);    // IDs retirés dans cette action
});
selection:cleared
Émis quand la sélection est entièrement vidée (en plus de selection:changed). Pratique pour réinitialiser un panneau de propriétés. 
voh.on('selection:cleared', function(data) {
    console.log('Sélection vidée sur canvas', data.canvasId);
    console.log('Anciens IDs:', data.previousIds);
});

Contour visuel (configurable par objet)

Chaque objet sélectionné affiche un contour rectangulaire autour de lui. L'apparence est configurable individuellement.

voh.objects.setSelectionBorderColor(objectId, color) → vrai/faux
Définit la couleur du contour de sélection. Défaut: 'rgba(0, 120, 215, 0.9)' (bleu).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
colortexteCouleur CSS (rgba, hex, rgb)
Retourne: vrai/faux
voh.objects.setSelectionBorderColor(objet, 'rgba(255, 0, 0, 0.9)');  // contour rouge
voh.objects.setSelectionBorderColor(objet, '#00ff00');                // contour vert
voh.objects.setSelectionBorderThickness(objectId, thickness) → vrai/faux
Définit l'épaisseur du contour de sélection en pixels. Défaut: 1.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
thicknessnombreÉpaisseur en pixels
Retourne: vrai/faux
voh.objects.setSelectionBorderThickness(objet, 3);  // contour plus épais
voh.objects.setSelectionBorderOffset(objectId, offset) → vrai/faux
Définit l'espace entre l'objet et son contour de sélection en pixels. Défaut: 0 (contour collé à l'extérieur de l'objet). Une valeur positive éloigne le contour, une valeur négative le fait rentrer par-dessus l'objet.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
offsetnombreEspace en pixels (positif = extérieur, 0 = collé, négatif = par-dessus l'objet)
Retourne: vrai/faux
voh.objects.setSelectionBorderOffset(objet, 5);   // contour avec 5px d'espace
voh.objects.setSelectionBorderOffset(objet, 0);   // contour collé à l'extérieur (défaut)
voh.objects.setSelectionBorderOffset(objet, -3);  // contour par-dessus l'objet (rentre de 3px)
Paramètres à la création: Les propriétés de contour peuvent aussi être passées directement à voh.objects.create(): selectionBorderColor, selectionBorderThickness, selectionBorderOffset, selectionBorderStyle.
voh.objects.setSelectionBorderStyle(objectId, style) → vrai/faux
Définit le style du contour de sélection. Défaut: 'solid'.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
styletexte'solid' (défaut), 'dashed' ou 'dotted'
Retourne: vrai/faux
voh.objects.setSelectionBorderStyle(objet, 'solid');   // trait continu (défaut)
voh.objects.setSelectionBorderStyle(objet, 'dashed');  // tirets
voh.objects.setSelectionBorderStyle(objet, 'dotted');  // pointillés

Mode de sélection par rectangle (par objet)

Chaque objet peut définir comment il réagit au rectangle de sélection: il doit être entièrement englobé ('enclosed') ou juste touché ('intersect').

voh.objects.setSelectionMode(objectId, mode) → vrai/faux
Définit le mode de sélection par rectangle pour cet objet.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
modetexte | null 'enclosed' (défaut) = l'objet doit être entièrement dans le rectangle
'intersect' = l'objet est sélectionné dès qu'il touche le rectangle
null = converti automatiquement en 'enclosed'
Retourne: vrai/faux
// Petit objet: sélectionné dès qu'il est touché
voh.objects.setSelectionMode(bouton, 'intersect');

// Grand objet de fond: sélectionné uniquement si entièrement englobé
voh.objects.setSelectionMode(fond, 'enclosed');

// Remettre au défaut: null est accepté et converti en 'enclosed'
voh.objects.setSelectionMode(objet, null);
console.log(voh.objects.getSelectionMode(objet));  // 'enclosed' (jamais null)
Paramètre à la création: Peut aussi être passé à voh.objects.create() via selectionMode: 'intersect'.

Comportements automatiques

  • Changement de page — La sélection est vidée automatiquement quand on change de page active (voh.canvas.pages.setActive).
  • Objet supprimé — Un objet supprimé (voh.objects.delete) est retiré silencieusement de la sélection.
  • Objet rendu invisible — Un objet rendu invisible (voh.objects.setVisible(id, false)) est désélectionné (avec événement).
  • Canvas supprimé — Les données de sélection du canevas sont nettoyées.
