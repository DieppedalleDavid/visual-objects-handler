voh.objects — Opacité

Gérer la transparence progressive d'un objet (0.0 à 1.0).

Définir l'opacité

voh.objects.setOpacity(objectId, opacity) → vrai/faux
Définit l'opacité de l'objet. La valeur est automatiquement bornée entre 0.0 (totalement transparent) et 1.0 (totalement opaque). Contrairement à setVisible() qui est tout ou rien, l'opacité permet une transparence graduelle.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
opacitynombreOpacité de 0.0 à 1.0
Retourne: vrai/faux
voh.objects.setOpacity(objet, 0.5);  // semi-transparent
voh.objects.setOpacity(objet, 1.0);  // opaque

Lire l'opacité

voh.objects.getOpacity(objectId) → nombre | null
Retourne l'opacité actuelle de l'objet (entre 0.0 et 1.0).
Retourne: nombre | null
console.log(voh.objects.getOpacity(objet));  // 1.0
