Événements — Objet

Les 33 types d'événements émis par les objets : 20 événements de propriétés + 9 événements d'interaction souris + 4 événements de drag. Tous sont captés par voh.on().

object:click

Émis quand l'utilisateur clique sur un objet (gauche, milieu ou droit). Le hit-test utilise une grille spatiale multi-niveaux pour identifier l'objet sous le curseur en O(1) même avec 20 000+ objets. L'objet avec le z-order le plus élevé a la priorité.
PropriétéTypeDescription
objectIdnombreL'ID de l'objet cliqué
canvasIdnombreL'ID du canevas
xnombrePosition X du clic (relative au canevas)
ynombrePosition Y du clic (relative au canevas)
buttontexte'left', 'middle' ou 'right'
originalEventMouseEventL'événement DOM natif (pour accéder à ctrlKey, shiftKey, etc.)
voh.on('object:click', (data) => {
    console.log('Clic sur objet', data.objectId, 'bouton:', data.button);
    console.log('Position:', data.x, data.y);
    if (data.originalEvent.ctrlKey) {
        console.log('Ctrl+clic !');
    }
});

object:mouseenter / object:mouseleave

Émis quand la souris entre dans un objet ou le quitte. Fonctionne comme les événements DOM mouseenter/mouseleave : un seul émis à l'entrée, un seul à la sortie (pas de bubbling). Si l'objet a un curseur défini via setCursor(), il est appliqué automatiquement au mouseenter et restauré au mouseleave.
PropriétéTypeDescription
objectIdnombreL'ID de l'objet
canvasIdnombreL'ID du canevas
xnombrePosition X de la souris
ynombrePosition Y de la souris
originalEventMouseEventL'événement DOM natif
voh.on('object:mouseenter', (data) => {
    console.log('Survol de', voh.objects.getName(data.objectId));
});
voh.on('object:mouseleave', (data) => {
    console.log('Quitte', voh.objects.getName(data.objectId));
});

object:dragstart

Émis une fois au démarrage du drag (premier mouvement de souris après l'enfoncement du bouton). C'est ici qu'on peut changer le curseur (setCursor) ou préparer l'interface.
PropriétéTypeDescription
objectIdnombreL'ID de l'objet dragué
canvasIdnombreL'ID du canevas
xnombrePosition X de la souris au démarrage
ynombrePosition Y de la souris au démarrage
buttontexteBouton utilisé : 'left', 'middle' ou 'right'
originalEventMouseEventL'événement DOM natif
voh.on('object:dragstart', (data) => {
    if (voh.objects.getCursor(data.objectId) === 'grab')
        voh.objects.setCursor(data.objectId, 'grabbing');
});

object:dragmove

Émis à chaque mouvement de souris pendant le drag (haute fréquence). En mode 'direct', l'objet a déjà été déplacé — on peut lire et corriger sa position (ex: contrainte dans une zone). En mode 'preview', le fantôme a déjà été repositionné.
PropriétéTypeDescription
objectIdnombreL'ID de l'objet
canvasIdnombreL'ID du canevas
xnombrePosition X actuelle de la souris
ynombrePosition Y actuelle de la souris
deltaXnombreDéplacement X depuis le dernier événement
deltaYnombreDéplacement Y depuis le dernier événement
originalEventMouseEventL'événement DOM natif
voh.on('object:dragmove', (data) => {
    // Contraindre l'objet dans le canevas (mode direct)
    const x = voh.objects.getX(data.objectId);
    const y = voh.objects.getY(data.objectId);
    if (x < 0) voh.objects.setX(data.objectId, 0);
    if (y < 0) voh.objects.setY(data.objectId, 0);
});

object:dragend

Émis une fois quand le bouton souris est relâché. x / y = position finale. previousX / previousY = position avant le drag. L'historique enregistre l'entrée à ce moment (une seule entrée par drag).
PropriétéTypeDescription
objectIdnombreL'ID de l'objet
canvasIdnombreL'ID du canevas
xnombrePosition X finale de l'objet
ynombrePosition Y finale de l'objet
previousXnombrePosition X avant le drag
previousYnombrePosition Y avant le drag
originalEventMouseEventL'événement DOM natif
voh.on('object:dragend', (data) => {
    console.log('Déplacé de', data.previousX, data.previousY, '→', data.x, data.y);
    if (voh.objects.getCursor(data.objectId) === 'grabbing')
        voh.objects.setCursor(data.objectId, 'grab');
});

object:dragcancel

Émis quand un drag est annulé par Escape ou par setMouseDragEnabled(false). L'objet a déjà été ramené à sa position d'origine. Aucune entrée n'est ajoutée à l'historique.
PropriétéTypeDescription
objectIdnombreL'ID de l'objet
canvasIdnombreL'ID du canevas
originalXnombrePosition X de départ (position restaurée)
originalYnombrePosition Y de départ (position restaurée)
voh.on('object:dragcancel', (data) => {
    console.log('Annulé, retour à', data.originalX, data.originalY);
    if (voh.objects.getCursor(data.objectId) === 'grabbing')
        voh.objects.setCursor(data.objectId, 'grab');
});
