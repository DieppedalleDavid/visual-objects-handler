voh.objects — Drag souris

Rendre un objet déplaçable à la souris. Deux modes sont disponibles: direct (l'objet suit la souris en temps réel) et preview (un fantôme suit la souris, l'objet saute à la position finale au relâchement).

Activer / désactiver le drag

voh.objects.setMouseDragEnabled(objectId, isEnabled) → vrai/faux
Active ou désactive le déplacement à la souris sur un objet. Si un drag est en cours au moment de l'appel avec false, il est annulé immédiatement (comportement identique à la touche Escape: l'objet retourne à sa position d'origine, aucune entrée n'est ajoutée à l'historique).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
isEnabledvrai/fauxtrue pour activer le drag
Retourne: vrai/faux
voh.objects.setMouseDragEnabled(objet, true);   // drag activé
voh.objects.setMouseDragEnabled(objet, false);  // drag désactivé (annule le drag en cours)
voh.objects.getMouseDragEnabled(objectId) → vrai/faux
Retourne true si le drag est activé sur l'objet.
Retourne: vrai/faux
console.log(voh.objects.getMouseDragEnabled(objet));  // true ou false

Bouton souris

voh.objects.setMouseDragButton(objectId, button) → vrai/faux
Définit le bouton souris qui déclenche le drag. Si le bouton 'right' est configuré, le menu contextuel du navigateur est automatiquement bloqué quand la souris est sur cet objet.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
buttontexte'left' (défaut), 'middle' ou 'right'
Retourne: vrai/faux
voh.objects.setMouseDragButton(objet, 'left');    // bouton gauche (défaut)
voh.objects.setMouseDragButton(objet, 'right');   // bouton droit (menu contextuel bloqué)
voh.objects.setMouseDragButton(objet, 'middle');  // molette enfoncée
voh.objects.getMouseDragButton(objectId) → texte
Retourne le bouton souris configuré pour le drag ('left', 'middle' ou 'right').
Retourne: texte
console.log(voh.objects.getMouseDragButton(objet));  // 'left', 'middle' ou 'right'

Mode de drag

voh.objects.setMouseDragMode(objectId, mode) → vrai/faux
Définit le comportement visuel du drag.

'direct' (défaut): l'objet se déplace en temps réel sous la souris pendant le drag. setX() / setY() sont appelés à chaque mouvement. L'historique enregistre une seule entrée par drag (position avant → position après), pas un undo par pixel.

'preview': l'objet reste immobile pendant le drag. Un sprite fantôme semi-transparent suit la souris. Au relâchement du bouton, l'objet saute à la position finale. Le fantôme est dessiné via mouseDragPreviewCallback (ou en pointillés bleus par défaut).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
modetexte'direct' (défaut) ou 'preview'
Retourne: vrai/faux
voh.objects.setMouseDragMode(objet, 'direct');   // l'objet suit la souris en temps réel
voh.objects.setMouseDragMode(objet, 'preview');  // fantôme suit la souris, l'objet saute à la fin
voh.objects.getMouseDragMode(objectId) → texte
Retourne le mode de drag de l'objet ('direct' ou 'preview').
Retourne: texte
console.log(voh.objects.getMouseDragMode(objet));  // 'direct' ou 'preview'

Décalage du drag (offset)

voh.objects.setMouseDragOffsetX(objectId, value) → vrai/faux
Définit le décalage horizontal du drag en pixels.
Avec 0 (défaut), l'objet garde sa position relative sous le curseur au moment du clic. Une valeur positive décale l'objet vers la droite au démarrage du drag.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
valuenombreDécalage X en pixels (0 = comportement naturel)
Retourne: vrai/faux
voh.objects.setMouseDragOffsetX(objet, 0);   // comportement naturel (défaut)
voh.objects.setMouseDragOffsetX(objet, 40);  // décale l'objet de 40px vers la droite au démarrage
voh.objects.getMouseDragOffsetX(objectId) → nombre
Retourne le décalage X du drag en pixels.
Retourne: nombre
console.log(voh.objects.getMouseDragOffsetX(objet));  // 0, 40, -20, etc.
voh.objects.setMouseDragOffsetY(objectId, value) → vrai/faux
Définit le décalage vertical du drag en pixels. Avec 0 (défaut), l'objet garde sa position relative sous le curseur. Une valeur positive décale l'objet vers le bas.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
valuenombreDécalage Y en pixels (0 = comportement naturel)
Retourne: vrai/faux
voh.objects.setMouseDragOffsetY(objet, 0);   // comportement naturel (défaut)
voh.objects.setMouseDragOffsetY(objet, 20);  // décale l'objet de 20px vers le bas au démarrage
voh.objects.getMouseDragOffsetY(objectId) → nombre
Retourne le décalage Y du drag en pixels.
Retourne: nombre
console.log(voh.objects.getMouseDragOffsetY(objet));  // 0, 20, -10, etc.

Callback du fantôme (mode preview)

voh.objects.setMouseDragPreviewCallback(objectId, callback) → vrai/faux
Définit la fonction de dessin du sprite fantôme utilisé en mode 'preview'. La signature est identique à drawingCallback: function(context, width, height, objectId).

Passer null pour utiliser le dessin par défaut: rectangle en pointillés bleus semi-transparents.

Ce changement est enregistré dans l'historique (Ctrl+Z fonctionne). Il prend effet au prochain drag.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
callbackfonction | nullFonction de dessin ou null pour le fantôme par défaut
Retourne: vrai/faux
// Fantôme par défaut (pointillés bleus)
voh.objects.setMouseDragPreviewCallback(objet, null);

// Fantôme personnalisé
voh.objects.setMouseDragPreviewCallback(objet, function(ctx, w, h, objectId) {
    ctx.strokeStyle = 'rgba(255, 100, 0, 0.8)';
    ctx.lineWidth = 2;
    ctx.setLineDash([6, 3]);
    ctx.strokeRect(1, 1, w - 2, h - 2);
    ctx.setLineDash([]);
    ctx.fillStyle = 'rgba(255, 100, 0, 0.15)';
    ctx.fillRect(0, 0, w, h);
});
voh.objects.getMouseDragPreviewCallback(objectId) → fonction | null
Retourne le callback du fantôme, ou null si le dessin par défaut est utilisé.
Retourne: fonction | null
const cb = voh.objects.getMouseDragPreviewCallback(objet);
console.log(cb !== null ? 'Fantôme personnalisé' : 'Fantôme par défaut');

Multi-drag (déplacement en multi-sélection)

Quand plusieurs objets sont sélectionnés et qu'on drag l'un d'eux, tous les sélectionnés bougent ensemble. Par défaut, chaque objet utilise son propre mouseDragMode, mais on peut configurer un mode différent pour le multi-drag avec les paramètres ci-dessous.

Chaîne de fallback:
Mode: mouseDragMultiMode → si null → mouseDragMode
Callback: mouseDragMultiPreviewCallback → si null → mouseDragPreviewCallback → si null → fantôme bleu par défaut

voh.objects.setMouseDragMultiMode(objectId, mode) → vrai/faux
Définit le mode de déplacement de cet objet quand il est dragué en multi-sélection. null (défaut): hérite de mouseDragMode.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
modetexte | null'direct', 'preview' ou null (hérite du solo)
Retourne: vrai/faux
// En solo: direct. En multi: fantôme.
voh.objects.setMouseDragMode(objet, 'direct');
voh.objects.setMouseDragMultiMode(objet, 'preview');

// Hériter du mode solo (comportement par défaut)
voh.objects.setMouseDragMultiMode(objet, null);
voh.objects.getMouseDragMultiMode(objectId) → texte | null
Retourne le mode multi-drag, ou null si l'objet hérite du mode solo.
Retourne: texte | null
console.log(voh.objects.getMouseDragMultiMode(objet));  // 'direct', 'preview' ou null
voh.objects.setMouseDragMultiPreviewCallback(objectId, callback) → vrai/faux
Définit le callback de dessin du fantôme pour le mode multi-sélection. null (défaut): hérite de mouseDragPreviewCallback. Même signature: function(context, width, height, objectId).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
callbackfonction | nullCallback de dessin du fantôme multi, ou null
Retourne: vrai/faux
// Fantôme rouge en multi pour indiquer "non déplaçable en groupe"
voh.objects.setMouseDragMultiPreviewCallback(objet, function(ctx, w, h) {
    ctx.fillStyle = 'rgba(255, 0, 0, 0.15)';
    ctx.fillRect(0, 0, w, h);
    ctx.strokeStyle = 'rgba(255, 0, 0, 0.8)';
    ctx.lineWidth = 2;
    ctx.setLineDash([6, 3]);
    ctx.strokeRect(1, 1, w - 2, h - 2);
});
voh.objects.getMouseDragMultiPreviewCallback(objectId) → fonction | null
Retourne le callback du fantôme multi, ou null si hérite du callback solo.
Retourne: fonction | null
console.log(voh.objects.getMouseDragMultiPreviewCallback(objet));

Événements du drag

Quatre événements sont émis pendant le cycle de vie d'un drag. En multi-drag, isMultiDrag: true est inclus dans les données. Voir Événements objet pour le détail complet.

ÉvénementDonnées reçuesQuand
object:dragstartobjectId, canvasId, x, y, button, originalEventLe drag commence
object:dragmoveobjectId, canvasId, x, y, deltaX, deltaY, isMultiDrag, originalEventChaque mouvement de souris pendant le drag
object:dragendobjectId, canvasId, x, y, previousX, previousY, isMultiDrag, originalEventBouton relâché (position finale)
object:dragcancelobjectId, canvasId, originalX, originalY, isMultiDragDrag annulé (Escape ou setMouseDragEnabled(false))

Comportements importants

Historique undo/redo: En mode 'direct', l'historique est mis en pause au début du drag et reprend à la fin. Une seule entrée est créée par drag (position de départ → position finale), même si l'objet a parcouru des centaines de pixels. Ctrl+Z annule tout le déplacement en une seule fois.
Annulation avec Escape: Pendant un drag, appuyer sur Escape annule le déplacement. L'objet retourne à sa position de départ. Aucune entrée n'est ajoutée à l'historique. L'événement object:dragcancel est émis. Appeler setMouseDragEnabled(false) pendant un drag produit le même effet.
Menu contextuel: Si mouseDragButton vaut 'right', le menu contextuel du navigateur est automatiquement bloqué sur cet objet, y compris pendant un drag bouton droit en cours.
Curseur pendant le drag: Le curseur de l'objet sous la souris est gelé pendant toute la durée du drag — passer sur d'autres objets ne change pas le curseur. Au relâchement, le curseur de l'objet se trouvant sous la souris à ce moment est appliqué.

Exemple complet

const objet = voh.objects.create(canvas, {
    name: 'Draggable',
    x: 100, y: 100, width: 200, height: 100,
    backgroundColor: '#3478d7',
    cursor: 'grab',
    mouseDragEnabled: true,
    mouseDragButton: 'left',
    mouseDragMode: 'direct'
});

// Changer le curseur pendant le drag
voh.on('object:dragstart', (data) => {
    if (voh.objects.getCursor(data.objectId) === 'grab')
        voh.objects.setCursor(data.objectId, 'grabbing');
});
voh.on('object:dragend', (data) => {
    if (voh.objects.getCursor(data.objectId) === 'grabbing')
        voh.objects.setCursor(data.objectId, 'grab');
});
voh.on('object:dragcancel', (data) => {
    if (voh.objects.getCursor(data.objectId) === 'grabbing')
        voh.objects.setCursor(data.objectId, 'grab');
});

// Logger la position finale
voh.on('object:dragend', (data) => {
    console.log('Déplacé de', data.previousX, data.previousY,
                '→', data.x, data.y);
});
Voir aussi: Le callback de dessin de l'objet lui-même (drawingCallback) est documenté dans la page Callback de dessin. Ne pas confondre avec mouseDragPreviewCallback qui dessine le fantôme pendant le drag.
