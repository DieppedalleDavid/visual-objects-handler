voh.objects — Z-order

Gérer l'ordre d'empilement (couches d'affichage) des objets sur un canevas.

Lire le z-order

voh.objects.getZOrder(objectId) → nombre | null
Retourne le z-order actuel de l'objet. Plus la valeur est élevée, plus l'objet est au-dessus des autres. Le z-order est limité aux objets du même canevas.
Retourne: nombre | null
console.log(voh.objects.getZOrder(objet));  // ex: 3

Définir le z-order

voh.objects.setZOrder(objectId, zOrder) → vrai/faux
Définit le z-order de l'objet à une valeur précise (nombre entier). Si un autre objet du même canevas a déjà ce z-order, tous les objets ayant un z-order supérieur ou égal sont décalés de +1 pour libérer la place. Chaque objet a toujours un z-order unique.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
zOrdernombreValeur de z-order souhaitée (nombre entier)
Retourne: vrai/faux
voh.objects.setZOrder(objet, 10);  // place l'objet au z-order 10

Amener au premier plan

voh.objects.bringToFront(objectId) → vrai/faux
Place l'objet au-dessus de tous les autres objets du même canevas.
Retourne: vrai/faux
voh.objects.bringToFront(objet);  // passe au-dessus de tout

Envoyer en arrière-plan

voh.objects.sendToBack(objectId) → vrai/faux
Place l'objet derrière tous les autres objets du même canevas.
Retourne: vrai/faux
voh.objects.sendToBack(objet);  // passe derrière tout

Monter d'un cran

voh.objects.moveUp(objectId) → vrai/faux
Monte l'objet d'un cran en échangeant son z-order avec l'objet juste au-dessus. Si l'objet est déjà au sommet, rien ne se passe.
Retourne: vrai/faux

Descendre d'un cran

voh.objects.moveDown(objectId) → vrai/faux
Descend l'objet d'un cran en échangeant son z-order avec l'objet juste en dessous. Si l'objet est déjà tout en bas, rien ne se passe.
Retourne: vrai/faux
voh.objects.moveUp(objet);    // monte d'un cran
voh.objects.moveDown(objet);  // descend d'un cran
