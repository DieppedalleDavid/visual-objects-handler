voh.objects — Création / Suppression

Créer des objets sur un canevas et les supprimer.

Créer un objet

voh.objects.create(canvasId, options) → nombre
Crée un nouvel objet sur un canevas et retourne son ID (un nombre unique). Cet ID est ensuite utilisé dans toutes les autres fonctions pour désigner cet objet. Par défaut, l'objet est un rectangle gris clair (objet nu). L'apparence peut être entièrement personnalisée via un callback de dessin (drawingCallback). Chaque objet créé reçoit un z-order croissant (le dernier créé est au-dessus).
ParamètreTypeDescription
canvasIdnombreL'ID du canevas sur lequel créer l'objet
nametexteNom de l'objet (par défaut: "Objet N")
xnombrePosition X sur le canevas en pixels (par défaut: 50)
ynombrePosition Y sur le canevas en pixels (par défaut: 50)
widthnombreLargeur en pixels (par défaut: 150, minimum: 5)
heightnombreHauteur en pixels (par défaut: 100, minimum: 5)
backgroundColorcouleurCouleur de fond du rendu par défaut (par défaut: gris clair)
isLockedvrai/fauxVerrouillé dès la création (par défaut: false)
isVisiblevrai/fauxVisible dès la création (par défaut: true)
opacitynombreOpacité: 0.0 = transparent → 1.0 = opaque (par défaut: 1.0)
drawingCallbackfonctionCallback de dessin personnalisé (par défaut: null = rendu par défaut)
cursortexteCurseur CSS au survol de l'objet (par défaut: null = curseur du canevas). Ex: 'pointer', 'grab', 'move'
mouseDragEnabledvrai/fauxActiver le drag de cet objet (par défaut: false)
mouseDragButtontexteBouton souris pour le drag: 'left', 'middle' ou 'right' (par défaut: 'left')
mouseDragModetexteMode de drag solo: 'direct' (suit la souris) ou 'preview' (fantôme) (par défaut: 'direct')
mouseDragOffsetXnombreDécalage X au début du drag en pixels (par défaut: 0)
mouseDragOffsetYnombreDécalage Y au début du drag en pixels (par défaut: 0)
mouseDragPreviewCallbackfonctionCallback de dessin du fantôme en mode preview (par défaut: null = fantôme bleu)
mouseDragMultiModetexteMode de drag en multi-sélection (par défaut: null = hérite de mouseDragMode)
mouseDragMultiPreviewCallbackfonctionCallback du fantôme en multi-sélection (par défaut: null = hérite de mouseDragPreviewCallback)
selectionBorderColorcouleurCouleur du contour de sélection (par défaut: 'rgba(0, 120, 215, 0.9)')
selectionBorderThicknessnombreÉpaisseur du contour de sélection en pixels (par défaut: 1)
selectionBorderOffsetnombreEspace entre l'objet et le contour en pixels (par défaut: 0 = collé à l'extérieur). Négatif = par-dessus l'objet
selectionBorderStyletexteStyle du contour: 'solid', 'dashed' ou 'dotted' (par défaut: 'solid')
selectionModetexteMode de sélection par rectangle: 'enclosed' (défaut, entièrement contenu), 'intersect' (touché suffit) ou null (converti en 'enclosed')
Retourne: nombre — L'ID de l'objet créé (nombre ≥ 1), ou -1 si le canevas n'existe pas.
// Objet avec tous les paramètres (valeurs par défaut)
const objet = voh.objects.create(canvas, {
    name:                         'Mon objet',
    x: 100,  y: 80,
    width: 200,  height: 120,
    backgroundColor:              'rgba(200, 200, 200, 1.0)',
    isLocked:                     false,
    isVisible:                    true,
    opacity:                      1.0,
    drawingCallback:              null,
    cursor:                       null,
    mouseDragEnabled:             false,
    mouseDragButton:              'left',
    mouseDragMode:                'direct',
    mouseDragOffsetX:             0,
    mouseDragOffsetY:             0,
    mouseDragPreviewCallback:     null,
    mouseDragMultiMode:           null,
    mouseDragMultiPreviewCallback: null,
    selectionBorderColor:         'rgba(0, 120, 215, 0.9)',
    selectionBorderThickness:     1,
    selectionBorderOffset:        0,
    selectionBorderStyle:         'solid',
    selectionMode:                'enclosed'
});

Supprimer un objet

voh.objects.delete(objectId) → vrai/faux
Supprime un objet du canevas. L'élément DOM est retiré et l'objet est supprimé de la mémoire.

Suppression en cascade: quand un canevas est supprimé via voh.canvas.delete(), tous ses objets sont automatiquement supprimés.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet à supprimer
Retourne: vrai/faux — true si l'objet a été supprimé, false si l'objet n'existe pas.
voh.objects.delete(objet);
console.log(voh.objects.exists(objet));  // false
