voh.objects — Informations

Compter, lister, vérifier l'existence et connaître le canevas parent d'un objet.

Compter les objets

voh.objects.getCount(canvasId?) → nombre
Retourne le nombre d'objets. Si canvasId est fourni, compte uniquement les objets de ce canevas. Si omis, retourne le nombre total d'objets dans l'instance.
Retourne: nombre
voh.objects.getCount(canvas);  // objets sur ce canevas
voh.objects.getCount();         // total de tous les canevas

Lister les objets

voh.objects.getList(canvasId?) → tableau
Retourne un tableau d'objets { id, name, canvasId }. Si canvasId est fourni, liste uniquement les objets de ce canevas. Si omis, liste tous les objets.
Retourne: Array<{ id, name, canvasId }>
const liste = voh.objects.getList(canvas);
for (const item of liste) {
    console.log(item.id, item.name);
}

Vérifier l'existence

voh.objects.exists(objectId) → vrai/faux
Retourne true si l'objet existe, false sinon.
Retourne: vrai/faux
voh.objects.exists(objet);  // true ou false

Canevas et page parent

voh.objects.getCanvasId(objectId) → nombre | null
Retourne l'ID du canevas contenant l'objet, ou null si l'objet n'existe pas.
Retourne: nombre | null
const id = voh.objects.getCanvasId(objet);
console.log('Canevas parent:', id);
voh.objects.getPageId(objectId) → nombre | null
Retourne l'ID de la page sur laquelle vit l'objet, ou null si l'objet n'existe pas. Un objet est créé sur la page active du canevas au moment de sa création.
Retourne: nombre | null
const pageId = voh.objects.getPageId(objet);
console.log('Page parente:', pageId);
