voh.objects — Curseur

Définir le curseur CSS affiché quand la souris survole un objet.

Définir le curseur

voh.objects.setCursor(objectId, cursor) → vrai/faux
Définit le curseur CSS affiché quand la souris survole l'objet. Passer null pour revenir au curseur par défaut du canevas. N'importe quelle valeur CSS valide est acceptée (voir tableau ci-dessous).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
cursortexte | nullCurseur CSS ou null pour le défaut
Retourne: vrai/faux
voh.objects.setCursor(objet, 'pointer');    // main de clic
voh.objects.setCursor(objet, 'move');       // croix de déplacement
voh.objects.setCursor(objet, null);         // curseur par défaut

Curseurs disponibles

Tous les curseurs CSS standard sont acceptés. Voici la liste complète :

ValeurDescription
Général
'default'Curseur par défaut du système (flèche)
'none'Curseur invisible
'auto'Le navigateur choisit selon le contexte
Liens et actions
'pointer'Main avec index pointé (lien, bouton cliquable)
'context-menu'Menu contextuel disponible
'help'Aide disponible (point d'interrogation)
'progress'En cours de traitement (sablier + flèche)
'wait'En attente (sablier seul)
Sélection
'cell'Sélection de cellule (tableau)
'crosshair'Réticule / viseur (dessin, sélection précise)
'text'Curseur texte (I-beam horizontal)
'vertical-text'Curseur texte vertical (I-beam vertical)
Glisser-déposer
'alias'Créer un alias / raccourci
'copy'Copier un élément
'move'Déplacer un élément (croix à 4 flèches)
'no-drop'Dépôt interdit ici
'not-allowed'Action interdite (cercle barré)
'grab'Main ouverte (saisir un objet)
'grabbing'Main fermée (objet saisi, en cours de drag)
Redimensionnement
'e-resize'Redimensionner vers l'est (droite)
'n-resize'Redimensionner vers le nord (haut)
'ne-resize'Redimensionner vers le nord-est (haut-droite)
'nw-resize'Redimensionner vers le nord-ouest (haut-gauche)
's-resize'Redimensionner vers le sud (bas)
'se-resize'Redimensionner vers le sud-est (bas-droite)
'sw-resize'Redimensionner vers le sud-ouest (bas-gauche)
'w-resize'Redimensionner vers l'ouest (gauche)
'ew-resize'Redimensionner horizontalement (gauche-droite)
'ns-resize'Redimensionner verticalement (haut-bas)
'nesw-resize'Redimensionner en diagonale (haut-droite / bas-gauche)
'nwse-resize'Redimensionner en diagonale (haut-gauche / bas-droite)
'col-resize'Redimensionner une colonne

Zoom
'zoom-in'Zoom avant
'zoom-out'Zoom arrière
Curseur personnalisé : Il est aussi possible d'utiliser une image comme curseur via la syntaxe CSS : 'url("mon-curseur.png"), auto'. La taille recommandée est 32×32 px. Le deuxième paramètre (auto) est le curseur de repli si l'image ne se charge pas.

Lire le curseur

voh.objects.getCursor(objectId) → texte | null
Retourne le curseur CSS de l'objet, ou null si aucun curseur personnalisé n'est défini.
Retourne: texte | null
console.log(voh.objects.getCursor(objet));  // 'pointer' ou null

Hit-test

voh.objects.hitTest(canvasId, x, y) → nombre | null
Recherche l'objet le plus haut (z-order maximum) sous le point (x, y) sur un canevas. Utilise la grille spatiale multi-niveaux pour des performances O(1) même avec 20 000+ objets. Retourne l'ID de l'objet trouvé, ou null si aucun objet n'est sous ce point.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
xnombrePosition X relative au canevas (pixels)
ynombrePosition Y relative au canevas (pixels)
Retourne: nombre | null — L'ID de l'objet ou null.
const objectId = voh.objects.hitTest(canvas, 150, 200);
if (objectId !== null) {
    console.log('Objet trouvé:', voh.objects.getName(objectId));
}
