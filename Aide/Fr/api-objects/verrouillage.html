voh.objects — Verrouillage

Verrouiller un objet pour empêcher sa manipulation à la souris (le code reste opérationnel).

Verrouiller / Déverrouiller

voh.objects.setLocked(objectId, locked) → vrai/faux
Verrouille (true) ou déverrouille (false) l'objet. Un objet verrouillé ne peut pas être manipulé à la souris (déplacement, redimensionnement), mais reste modifiable par le code (setX, setWidth, etc.).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
lockedvrai/fauxtrue = verrouillé, false = déverrouillé
Retourne: vrai/faux
voh.objects.setLocked(objet, true);   // verrouille
voh.objects.setLocked(objet, false);  // déverrouille

Lire l'état

voh.objects.isLocked(objectId) → vrai/faux | null
Retourne true si l'objet est verrouillé, false s'il ne l'est pas, ou null si l'objet n'existe pas.
Retourne: vrai/faux | null
if (voh.objects.isLocked(objet)) {
    console.log('L\'objet est verrouillé.');
}
voh.objects.getLocked(objectId) → vrai/faux | null
Alias de isLocked(). Retourne la même valeur. Utile pour la cohérence avec les autres getters (getName, getWidth, etc.).
Retourne: vrai/faux | null
voh.objects.getLocked(objet);  // identique à isLocked()
