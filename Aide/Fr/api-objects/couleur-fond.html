voh.objects — Couleur de fond

Changer la couleur de fond du rendu par défaut d'un objet. Par défaut, un objet sans couleur définie apparaît en gris clair (rgba(200, 200, 200, 1.0)).

Définir la couleur

voh.objects.setBackgroundColor(objectId, color) → vrai/faux
Définit la couleur de fond de l'objet. Accepte tous les formats CSS: rgba(), rgb(), #hex, noms de couleurs.

Note: cette couleur est utilisée uniquement par le rendu par défaut (rectangle plein). Si un drawingCallback est défini, cette propriété n'a aucun effet visuel (le callback dessine ce qu'il veut).
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
colorcouleurCouleur CSS
Retourne: vrai/faux
voh.objects.setBackgroundColor(objet, 'rgba(220, 60, 60, 1.0)');
voh.objects.setBackgroundColor(objet, '#4CAF50');
voh.objects.setBackgroundColor(objet, 'steelblue');

Lire la couleur

voh.objects.getBackgroundColor(objectId) → texte | null
Retourne la couleur de fond de l'objet. Par défaut: rgba(200, 200, 200, 1.0) (gris clair).
Retourne: texte | null
console.log(voh.objects.getBackgroundColor(objet));  // "rgba(200, 200, 200, 1.0)"
