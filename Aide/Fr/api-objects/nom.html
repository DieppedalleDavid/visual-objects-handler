voh.objects — Nom

Définir et lire le nom d'un objet.

Définir le nom

voh.objects.setName(objectId, name) → vrai/faux
Définit le nom de l'objet. Le nom est libre et sert à identifier l'objet dans les logs et l'interface.
ParamètreTypeDescription
objectIdnombreL'ID de l'objet
nametexteLe nouveau nom
Retourne: vrai/faux — true si réussi.
voh.objects.setName(objet, 'Bouton principal');

Lire le nom

voh.objects.getName(objectId) → texte | null
Retourne le nom de l'objet. Par défaut: "Objet N" (N = ID de l'objet).
Retourne: texte | null
console.log(voh.objects.getName(objet));  // "Bouton principal"
