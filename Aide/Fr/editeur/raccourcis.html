Raccourcis clavier

Liste complète des raccourcis clavier disponibles dans l'éditeur de code.

Les raccourcis utilisant Ctrl fonctionnent aussi avec Cmd sur macOS.

Fichier

RaccourciAction
Ctrl+NNouveau fichier (vide l'éditeur)
Ctrl+OOuvrir un fichier .js ou .ts depuis le disque
Ctrl+SEnregistrer le fichier courant
Ctrl+Shift+SEnregistrer sous (nouveau nom de fichier)

Édition

RaccourciAction
Ctrl+ZAnnuler la dernière modification
Ctrl+YRétablir la modification annulée
Ctrl+XCouper la sélection
Ctrl+CCopier la sélection
Ctrl+VColler depuis le presse-papiers
Ctrl+ATout sélectionner

Recherche et remplacement

RaccourciAction
Ctrl+FOuvrir le panneau de recherche
Ctrl+HOuvrir le panneau de recherche avec remplacement
Ctrl+GOccurrence suivante
Ctrl+Shift+GOccurrence précédente
EntréeOccurrence suivante (dans le panneau de recherche)
Shift+EntréeOccurrence précédente (dans le panneau de recherche)
ÉchapFermer le panneau de recherche

Le panneau de recherche propose trois options de filtrage : Respecter la casse, Mot entier seulement et Expression régulière. Le compteur d'occurrences s'affiche en temps réel.

Code

RaccourciAction
TabIndenter la sélection (ou insérer 4 espaces si pas de sélection)
Shift+TabDés-indenter la sélection ou la ligne courante
Ctrl+Shift+FReformater le code (indentation automatique)
Ctrl+QReplier / Déplier le bloc de code sous le curseur
Reformater le code corrige automatiquement l'indentation. Si du texte est sélectionné, seule la sélection est reformatée. Sinon, tout le code est reformaté.

Exécution et aide

RaccourciAction
F5Exécuter le script dans un nouvel onglet (runner.html)
F1Ouvrir la page de documentation de la fonction API sous le curseur

Le raccourci F1 fonctionne quand le curseur texte est positionné sur un nom de fonction API (par exemple voh.objects.create). La page de documentation correspondante s'ouvre dans un nouvel onglet du navigateur.

Autocomplétion

RaccourciAction
↑ / ↓Naviguer dans la liste de propositions
Entrée ou TabValider la proposition sélectionnée
ÉchapFermer la liste de propositions

L'autocomplétion se déclenche automatiquement en tapant. Voir la page Fonctionnalités de l'éditeur pour les détails.

Menu contextuel

Un clic droit dans l'éditeur ouvre un menu contextuel avec les actions courantes : Annuler, Couper, Copier, Coller, Tout sélectionner, Rechercher, Remplacer et Reformater le code.
