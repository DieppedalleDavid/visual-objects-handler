voh.canvas — Verrouillage

Un canvas verrouillé ne peut être ni déplacé, ni redimensionné par la souris. Les objets à l'intérieur restent manipulables (le verrou concerne le canvas, pas son contenu). L'API peut toujours modifier la position et la taille même si le canvas est verrouillé.

voh.canvas.setLocked(canvasId, isLocked) → vrai/faux
Verrouille ou déverrouille un canvas.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
isLockedvrai/fauxtrue = verrouillé, false = déverrouillé
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setLocked(canvas1, true);

// L'API fonctionne toujours même verrouillé
voh.canvas.setX(canvas1, 200);  // fonctionne
voh.canvas.getLocked(canvasId) → vrai/faux ou null
Retourne l'état de verrouillage.
Retourne: vrai/faux — true si verrouillé, false si déverrouillé, ou null si le canvas n'existe pas.
voh.canvas.isLocked(canvasId) → vrai/faux ou null
Alias de getLocked. Forme plus naturelle pour les booléens.
Retourne: vrai/faux — true si verrouillé, false si déverrouillé, ou null si le canvas n'existe pas.
if (voh.canvas.isLocked(canvas1)) {
    console.log('Le canvas est verrouillé');
}
