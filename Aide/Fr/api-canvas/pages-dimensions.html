voh.canvas.pages.dimensions — Dimensions par page

Chaque page peut avoir ses propres dimensions. Les dimensions sont stockées par page mais le rendu visuel (scroll) sera implémenté dans une prochaine version.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Largeur

voh.canvas.pages.dimensions.setWidth(canvasId, width, pageId?) → vrai/faux
Définit la largeur d'une page en pixels.
ParamètreTypeDescription
canvasIdnombreID du canvas
widthnombreLa valeur à définir
pageIdnombreID de la page (optionnel, défaut: page active)
Retourne: vrai/faux — true si réussi, false si canvas/page inexistant.
voh.canvas.pages.dimensions.setWidth(canvas, 1200);
voh.canvas.pages.dimensions.getWidth(canvasId, pageId?) → nombre | null
Retourne la largeur.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: nombre | null — la valeur, ou null si canvas/page inexistant.
console.log(voh.canvas.pages.dimensions.getWidth(canvas));
