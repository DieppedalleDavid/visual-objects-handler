voh.canvas — Opacité

L'opacité globale du canvas entier, de 0.0 (totalement transparent) à 1.0 (totalement opaque). Affecte tout le contenu du canvas (fond, objets, overlay).

voh.canvas.setOpacity(canvasId, opacity) → vrai/faux
Définit l'opacité d'un canvas. La valeur est automatiquement bornée entre 0.0 et 1.0.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
opacitynombreOpacité: 0.0 = transparent → 1.0 = opaque
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setOpacity(canvas1, 0.5);  // semi-transparent
voh.canvas.setOpacity(canvas1, 1.0);  // opaque
voh.canvas.getOpacity(canvasId) → nombre ou null
Retourne l'opacité actuelle du canvas.
Retourne: nombre — l'opacité (0.0 à 1.0), ou null si le canvas n'existe pas.
console.log(voh.canvas.getOpacity(canvas1));  // ex: 0.5
