voh.canvas — Création / Suppression

Créer des canvas dans une zone et les supprimer.

Créer un canvas

voh.canvas.create(zoneId, options) → nombre
Crée un nouveau canvas dans une zone et retourne son ID (un nombre unique). Cet ID est ensuite utilisé dans toutes les autres fonctions pour désigner ce canvas. Le premier canvas créé dans l'instance est automatiquement activé. Chaque canvas a un z-order croissant (le dernier créé est au-dessus des autres).
ParamètreTypeDescription
zoneIdnombreL'ID de la zone dans laquelle créer le canvas
nametexteNom du canvas (par défaut: "Canvas N")
xnombrePosition X dans la zone en pixels (par défaut: 50)
ynombrePosition Y dans la zone en pixels (par défaut: 50)
widthnombreLargeur en pixels (par défaut: 800)
heightnombreHauteur en pixels (par défaut: 600)
backgroundColorcouleurCouleur de fond (par défaut: blanc)
isLockedvrai/fauxVerrouillé dès la création (par défaut: false)
isVisiblevrai/fauxVisible dès la création (par défaut: true)
opacitynombreOpacité: 0.0 = transparent → 1.0 = opaque (par défaut: 1.0)
selectionEnabledvrai/fauxSélection à la souris activée (par défaut: false) — voir Sélection
selectionMouseButtontexteBouton souris: 'left', 'middle', 'right' (par défaut: 'left')
selectionBackgroundColorcouleurCouleur de fond du rectangle de sélection (par défaut: bleu semi-transparent)
selectionBorderColorcouleurCouleur de bordure du rectangle de sélection (par défaut: bleu)
selectionBorderThicknessnombreÉpaisseur de bordure du rectangle en pixels (par défaut: 1)
selectionBorderStyletexteStyle de trait: 'solid', 'dashed', 'dotted' (par défaut: 'solid')
Retourne: nombre — L'ID du canvas créé (nombre ≥ 1), ou -1 si la zone n'existe pas.
// Canvas simple
const canvas = voh.canvas.create(zone, {
    name: 'Mon canvas',
    x: 30,
    y: 30,
    width: 600,
    height: 400
});

Supprimer un canvas

voh.canvas.delete(canvasId) → vrai/faux
Supprime un canvas et tous ses objets. Si c'était le canvas actif, aucun autre canvas n'est activé automatiquement — voh.canvas.getActiveId() retournera null. C'est au développeur d'activer un autre canvas si nécessaire via voh.canvas.setActive().
ParamètreTypeDescription
canvasIdnombreL'ID du canvas à supprimer
Retourne: vrai/faux — true si le canvas a été supprimé, false si le canvas n'existe pas.
voh.canvas.delete(canvas);
console.log(voh.canvas.exists(canvas));    // false
console.log(voh.canvas.getActiveId());  // null si c'était le canvas actif
