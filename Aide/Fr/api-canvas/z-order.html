voh.canvas — Z-order

L'ordre d'empilement des canvas dans une zone. Un z-order plus élevé signifie que le canvas est affiché au-dessus des autres. Par défaut, chaque nouveau canvas a un z-order supérieur au précédent.

voh.canvas.bringToFront(canvasId) → vrai/faux
Place un canvas au premier plan (z-order le plus élevé).
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.bringToFront(canvas1);
voh.canvas.sendToBack(canvasId) → vrai/faux
Place un canvas en arrière-plan (z-order le plus bas).
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.sendToBack(canvas1);
voh.canvas.moveUp(canvasId) → vrai/faux
Monte un canvas d'un cran dans le z-order (échange avec le canvas juste au-dessus).
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.moveUp(canvas1);
voh.canvas.moveDown(canvasId) → vrai/faux
Descend un canvas d'un cran dans le z-order (échange avec le canvas juste en-dessous).
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.moveDown(canvas1);
voh.canvas.setZOrder(canvasId, zOrder) → vrai/faux
Définit le z-order du canvas à une valeur précise (nombre entier). Si un autre canvas de la même zone a déjà ce z-order, tous les canvas ayant un z-order supérieur ou égal sont décalés de +1 pour libérer la place.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
zOrdernombreValeur de z-order souhaitée (nombre entier)
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setZOrder(canvas1, 10);  // place le canvas au z-order 10
voh.canvas.getZOrder(canvasId) → nombre ou null
Retourne le z-order actuel d'un canvas.
Retourne: nombre — le z-order (entier ≥ 0), ou null si le canvas n'existe pas.
console.log(voh.canvas.getZOrder(canvas1));  // ex: 3
