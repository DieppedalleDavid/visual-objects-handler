voh.canvas.pages.selection — Sélection souris par page

Chaque page a ses propres réglages de rectangle de sélection souris. Au changement de page, les réglages sont sauvegardés et restaurés automatiquement.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Activation

voh.canvas.pages.selection.setEnabled(canvasId, enabled, pageId?) → vrai/faux
Active ou désactive la sélection à la souris (défaut: false).
ParamètreTypeDescription
canvasIdnombreID du canvas
enabledvrai/fauxLa valeur à définir
pageIdnombreID de la page (optionnel, défaut: page active)
Retourne: vrai/faux — true si réussi, false si canvas/page inexistant.
voh.canvas.pages.selection.setEnabled(canvas, true);
voh.canvas.pages.selection.getEnabled(canvasId, pageId?) → vrai/faux | null
Retourne l'état d'activation.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux | null — la valeur, ou null si canvas/page inexistant.
console.log(voh.canvas.pages.selection.getEnabled(canvas));
