voh.canvas.pages — Gestion des pages

Ajouter, supprimer, naviguer et organiser les pages d'un canvas.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Ajout / Suppression

voh.canvas.pages.add(canvasId, params?) → nombre
Ajoute une page au canvas. La page hérite des valeurs par défaut ou des paramètres fournis (name, width, height, backgroundColor, etc.).
ParamètreTypeDescription
canvasIdnombreID du canvas
paramsobjet{ name, width, height, backgroundColor, borderColor, ... }
Retourne: nombre — l'ID de la page créée, ou -1 si erreur.
// Ajouter une page avec les valeurs par défaut
const page2 = voh.canvas.pages.add(canvas);

// Ajouter une page personnalisée
const page3 = voh.canvas.pages.add(canvas, {
    name: 'Niveau 3',
    width: 1200, height: 900,
    backgroundColor: 'rgba(30, 30, 40, 1.0)'
});
voh.canvas.pages.remove(canvasId, pageId) → vrai/faux
Supprime une page. Impossible de supprimer la dernière page — il doit toujours en rester au moins une. Si la page supprimée était active, la page précédente (ou la première) est automatiquement activée.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page à supprimer
Retourne: vrai/faux — true si supprimée, false si dernière page ou inexistante.
voh.canvas.pages.remove(canvas, page3);  // supprime la page 3

Navigation

voh.canvas.pages.setActive(canvasId, pageId) → vrai/faux
Active une page (la rend courante). Le contenu visible du canvas change pour afficher cette page. Les objets de l'ancienne page sont masqués, ceux de la nouvelle apparaissent.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page à activer
Retourne: vrai/faux — true si activée.
voh.canvas.pages.setActive(canvas, page2);  // basculer sur la page 2
voh.canvas.pages.getActiveId(canvasId) → nombre | null
Retourne l'ID de la page actuellement active.
ParamètreTypeDescription
canvasIdnombreID du canvas
Retourne: nombre | null — l'ID de la page active.
console.log(voh.canvas.pages.getActiveId(canvas));  // 1
voh.canvas.pages.next(canvasId) → vrai/faux
Active la page suivante. Circulaire: de la dernière, on revient à la première.
Retourne: vrai/faux — true si réussi, false si une seule page.
voh.canvas.pages.next(canvas);  // page suivante
voh.canvas.pages.previous(canvasId) → vrai/faux
Active la page précédente. Circulaire: de la première, on passe à la dernière.
Retourne: vrai/faux — true si réussi, false si une seule page.
voh.canvas.pages.previous(canvas);  // page précédente
voh.canvas.pages.goTo(canvasId, index) → vrai/faux
Active la page à l'index donné (0-based).
ParamètreTypeDescription
canvasIdnombreID du canvas
indexnombreIndex de la page (0-based)
Retourne: vrai/faux — true si réussi, false si index hors limites.
voh.canvas.pages.goTo(canvas, 0);  // aller à la première page
voh.canvas.pages.goTo(canvas, 2);  // aller à la troisième page

Informations

voh.canvas.pages.getCount(canvasId) → nombre
Retourne le nombre total de pages du canvas.
ParamètreTypeDescription
canvasIdnombreID du canvas
Retourne: nombre — le nombre de pages.
console.log(voh.canvas.pages.getCount(canvas));  // 3
voh.canvas.pages.getList(canvasId) → tableau
Retourne la liste ordonnée des pages avec leur ID et nom.
ParamètreTypeDescription
canvasIdnombreID du canvas
Retourne: tableau — ex: [{ id: 1, name: 'Page 1' }, ...]
const liste = voh.canvas.pages.getList(canvas);
liste.forEach(p => console.log(p.id, p.name));
voh.canvas.pages.exists(canvasId, pageId) → vrai/faux
Vérifie si une page existe dans le canvas.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
Retourne: vrai/faux — true si la page existe.
console.log(voh.canvas.pages.exists(canvas, page2));  // true
voh.canvas.pages.getIndex(canvasId, pageId) → nombre
Retourne l'index (0-based) d'une page dans l'ordre du canvas.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
Retourne: nombre — l'index, ou -1 si introuvable.
console.log(voh.canvas.pages.getIndex(canvas, page1));  // 0

Nom

voh.canvas.pages.setName(canvasId, pageId, name) → vrai/faux
Définit le nom d'une page. Cette action est enregistrée dans l'historique undo/redo.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
nametexteNouveau nom
Retourne: vrai/faux — true si réussi.
voh.canvas.pages.setName(canvas, page1, 'Niveau principal');
voh.canvas.pages.getName(canvasId, pageId) → texte | null
Retourne le nom d'une page.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
Retourne: texte | null — le nom, ou null si erreur.
console.log(voh.canvas.pages.getName(canvas, page1));  // 'Niveau principal'

Ordre des pages

voh.canvas.pages.moveUp(canvasId, pageId) → vrai/faux
Monte une page d'un cran dans l'ordre.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
Retourne: vrai/faux — true si déplacée.
voh.canvas.pages.moveUp(canvas, page3);  // monte page3 d'un cran
voh.canvas.pages.moveDown(canvasId, pageId) → vrai/faux
Descend une page d'un cran dans l'ordre.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
Retourne: vrai/faux — true si déplacée.
voh.canvas.pages.moveDown(canvas, page1);  // descend page1 d'un cran
voh.canvas.pages.moveTo(canvasId, pageId, index) → vrai/faux
Déplace une page à un index précis (0-based).
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
indexnombreIndex cible (0-based)
Retourne: vrai/faux — true si déplacée.
voh.canvas.pages.moveTo(canvas, page3, 0);  // mettre page3 en première position

Duplication

voh.canvas.pages.duplicate(canvasId, pageId) → nombre
Duplique une page avec toutes ses propriétés. La copie est insérée juste après l'originale et son nom reçoit le suffixe " (copie)".
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page à dupliquer
Retourne: nombre — l'ID de la copie, ou -1 si erreur.
const copie = voh.canvas.pages.duplicate(canvas, page1);
console.log(voh.canvas.pages.getName(canvas, copie));  // 'Page 1 (copie)'

Export / Import

voh.canvas.pages.export(canvasId, pageId) → objet | null
Exporte une page en objet JSON sérialisable contenant toutes ses propriétés. L'ID et le canvasId ne sont pas inclus — ils seront régénérés à l'import.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page
Retourne: objet | null — les données JSON, ou null si erreur.
const donnees = voh.canvas.pages.export(canvas, page1);
console.log(JSON.stringify(donnees));
voh.canvas.pages.import(canvasId, data) → nombre
Importe une page dans le canvas depuis un objet JSON. Un nouvel ID est généré automatiquement.
ParamètreTypeDescription
canvasIdnombreID du canvas cible
dataobjetDonnées JSON de la page
Retourne: nombre — l'ID de la page importée, ou -1 si erreur.
// Exporter puis importer dans un autre canvas
const donnees = voh.canvas.pages.export(canvas1, page1);
const nouvellePage = voh.canvas.pages.import(canvas2, donnees);
Événements: les opérations sur les pages émettent page:added, page:removed, page:activated, page:renamed, page:moved, page:duplicated.
