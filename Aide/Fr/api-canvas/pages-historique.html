voh.canvas.pages.history — Undo/Redo par page

Chaque page possède son propre historique de modifications. Toutes les modifications de propriétés de page sont automatiquement enregistrées. Le système supporte le batch (pause/resume) pour grouper plusieurs modifications en un seul undo.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Annuler / Refaire

voh.canvas.pages.history.undo(canvasId, pageId?) → vrai/faux
Annule la dernière action. Le rendu visuel (fond, bordure, grille) est automatiquement mis à jour.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux — true si annulée, false si rien à annuler.
voh.canvas.pages.history.undo(canvas);  // annuler la dernière modification
voh.canvas.pages.history.redo(canvasId, pageId?) → vrai/faux
Refait la dernière action annulée. Le rendu visuel est automatiquement mis à jour.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux — true si refaite, false si rien à refaire.
voh.canvas.pages.history.redo(canvas);  // refaire
voh.canvas.pages.history.canUndo(canvasId, pageId?) → vrai/faux
Retourne true s'il y a au moins une action annulable.
Retourne: vrai/faux — vrai/faux
if (voh.canvas.pages.history.canUndo(canvas)) {
    voh.canvas.pages.history.undo(canvas);
}
voh.canvas.pages.history.canRedo(canvasId, pageId?) → vrai/faux
Retourne true s'il y a au moins une action refaisable.
Retourne: vrai/faux — vrai/faux
console.log('Peut refaire:', voh.canvas.pages.history.canRedo(canvas));

Compteurs

voh.canvas.pages.history.getUndoCount(canvasId, pageId?) → nombre
Retourne le nombre d'actions annulables.
Retourne: nombre — nombre
console.log('Undo:', voh.canvas.pages.history.getUndoCount(canvas));  // ex: 5
voh.canvas.pages.history.getRedoCount(canvasId, pageId?) → nombre
Retourne le nombre d'actions refaisables.
Retourne: nombre — nombre
console.log('Redo:', voh.canvas.pages.history.getRedoCount(canvas));  // ex: 2

Configuration

voh.canvas.pages.history.setMaxLevel(canvasId, maxLevel, pageId?) → vrai/faux
Définit le nombre maximum d'actions conservées dans la pile undo (défaut: 50, minimum: 1). Si la pile dépasse le nouveau max, les plus anciennes sont éjectées.
ParamètreTypeDescription
canvasIdnombreID du canvas
maxLevelnombreNombre maximum d'actions (minimum 1)
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux — true si réussi.
voh.canvas.pages.history.setMaxLevel(canvas, 100);  // garder 100 actions
voh.canvas.pages.history.getMaxLevel(canvasId, pageId?) → nombre | null
Retourne le niveau maximum de l'historique.
Retourne: nombre | null — le max, ou null si erreur.
console.log(voh.canvas.pages.history.getMaxLevel(canvas));  // 100
voh.canvas.pages.history.clear(canvasId, pageId?) → vrai/faux
Vide entièrement l'historique undo et redo de la page.
Retourne: vrai/faux — true si réussi.
voh.canvas.pages.history.clear(canvas);  // vider tout l'historique

Batch (pause / resume)

voh.canvas.pages.history.pause(canvasId, pageId?) → vrai/faux
Met l'historique en pause. Toutes les modifications effectuées pendant la pause seront regroupées en une seule action à l'appel de resume(). Les pauses sont imbriquables (compteur).
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux — true si réussi.
// 3 modifications = 1 seul undo
voh.canvas.pages.history.pause(canvas);
voh.canvas.pages.background.setColor(canvas, 'rgba(50, 50, 60, 1.0)');
voh.canvas.pages.border.setVisible(canvas, true);
voh.canvas.pages.border.setColor(canvas, 'rgba(255, 100, 0, 0.8)');
voh.canvas.pages.history.resume(canvas, null, 'Style sombre + bordure orange');

// Un seul undo annule les 3 modifications
voh.canvas.pages.history.undo(canvas);
voh.canvas.pages.history.resume(canvasId, pageId?, description?) → vrai/faux
Reprend l'enregistrement. Les actions accumulées pendant la pause deviennent une seule action batch. La description optionnelle nomme le batch.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
descriptiontexteDescription du batch (optionnel)
Retourne: vrai/faux — true si réussi, false si pas en pause.
voh.canvas.pages.history.resume(canvas, null, 'Changement de thème');
Événements: les opérations émettent page:undone, page:redone, page:historyCleared.
