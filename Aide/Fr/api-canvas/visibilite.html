voh.canvas — Visibilité

Un canvas masqué n'est pas rendu et ne reçoit pas d'événements souris. Ses données (objets, pages, etc.) restent en mémoire — il peut être réaffiché à tout moment.

voh.canvas.setVisible(canvasId, isVisible) → vrai/faux
Affiche ou masque un canvas.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
isVisiblevrai/fauxtrue = visible, false = masqué
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setVisible(canvas1, false);  // masquer
voh.canvas.setVisible(canvas1, true);   // réafficher
voh.canvas.getVisible(canvasId) → vrai/faux ou null
Retourne l'état de visibilité.
Retourne: vrai/faux — true si visible, false si masqué, ou null si le canvas n'existe pas.
voh.canvas.isVisible(canvasId) → vrai/faux ou null
Alias de getVisible. Forme plus naturelle pour les booléens.
Retourne: vrai/faux — true si visible, false si masqué, ou null si le canvas n'existe pas.
if (voh.canvas.isVisible(canvas1)) {
    console.log('Le canvas est affiché');
}
