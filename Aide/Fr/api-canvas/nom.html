voh.canvas — Nom

Chaque canvas a un nom personnalisable pour l'identifier.

voh.canvas.setName(canvasId, name) → vrai/faux
Définit le nom d'un canvas.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
nametexteLe nouveau nom
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setName(canvas1, 'Éditeur principal');
voh.canvas.getName(canvasId) → texte ou null
Retourne le nom d'un canvas.
Retourne: texte — le nom du canvas, ou null si le canvas n'existe pas.
console.log(voh.canvas.getName(canvas1));  // "Éditeur principal"
