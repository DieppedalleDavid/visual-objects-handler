voh.canvas — Informations

Obtenir des informations sur les canvas existants.

voh.canvas.getCount(zoneId) → nombre
Retourne le nombre de canvas dans une zone. Si zoneId n'est pas fourni (ou null), retourne le total de tous les canvas de l'instance.
Retourne: nombre — le nombre de canvas (0 si aucun).
console.log(voh.canvas.getCount(zone1));  // ex: 3
console.log(voh.canvas.getCount());       // total de l'instance
voh.canvas.getList(zoneId) → tableau de nombres
Retourne la liste des IDs de canvas dans une zone. Si zoneId n'est pas fourni, retourne tous les canvas de l'instance.
Retourne: tableau — les IDs des canvas (tableau vide [] si aucun).
console.log(voh.canvas.getList(zone1));  // ex: [1, 2, 3]
voh.canvas.exists(canvasId) → vrai/faux
Vérifie si un canvas existe encore (utile après une suppression).
Retourne: vrai/faux — true si le canvas existe, false sinon.
console.log(voh.canvas.exists(1));  // true ou false
voh.canvas.getZoneId(canvasId) → nombre ou null
Retourne l'ID de la zone à laquelle appartient un canvas.
Retourne: nombre — l'ID de la zone parente, ou null si le canvas n'existe pas.
console.log(voh.canvas.getZoneId(canvas1));  // ex: 1
