voh.canvas.pages.background — Fond par page

Chaque page a sa propre couleur de fond et image de fond optionnelle. Le fond est rendu par Pixi.js (GPU) et se met à jour en temps réel.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Couleur de fond

voh.canvas.pages.background.setColor(canvasId, color, pageId?) → vrai/faux
Définit la couleur de fond d'une page.
ParamètreTypeDescription
canvasIdnombreID du canvas
colorcouleur CSSLa valeur à définir
pageIdnombreID de la page (optionnel, défaut: page active)
Retourne: vrai/faux — true si réussi, false si canvas/page inexistant.
voh.canvas.pages.background.setColor(canvas, 'rgba(30, 30, 40, 1.0)');
voh.canvas.pages.background.getColor(canvasId, pageId?) → couleur CSS | null
Retourne la couleur de fond.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: couleur CSS | null — la valeur, ou null si canvas/page inexistant.
console.log(voh.canvas.pages.background.getColor(canvas));
