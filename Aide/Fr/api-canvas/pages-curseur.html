voh.canvas.pages.cursor — Curseur par page

Chaque page peut avoir son propre curseur souris. Au changement de page, le curseur est automatiquement mis à jour.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Curseur

voh.canvas.pages.cursor.set(canvasId, cursor, pageId?) → vrai/faux
Définit le curseur souris d'une page. Accepte toute valeur CSS cursor valide (défaut: 'default').
ParamètreTypeDescription
canvasIdnombreID du canvas
cursortexteValeur CSS cursor
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux — true si réussi.
voh.canvas.pages.cursor.set(canvas, 'crosshair');  // curseur en croix
voh.canvas.pages.cursor.set(canvas, 'pointer');    // curseur main
voh.canvas.pages.cursor.set(canvas, 'default');    // curseur par défaut
voh.canvas.pages.cursor.get(canvasId, pageId?) → texte | null
Retourne le curseur souris configuré pour la page.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: texte | null — la valeur CSS cursor, ou null si erreur.
console.log(voh.canvas.pages.cursor.get(canvas));  // 'crosshair'
Événements: les modifications émettent page:cursorChanged.
