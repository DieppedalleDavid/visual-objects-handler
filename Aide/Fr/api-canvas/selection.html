voh.canvas.selection — Rectangle de sélection

Propriétés visuelles et comportement du rectangle tracé à la souris pour sélectionner des objets sur un canevas. Chaque canevas possède ses propres réglages de sélection, indépendants des autres.

Ces propriétés contrôlent uniquement l'apparence du rectangle de sélection. Le comportement de la sélection (hit-test, multi-sélection, etc.) sera disponible dans une prochaine version.

Activation

voh.canvas.selection.setEnabled(canvasId, enabled) → vrai/faux
Active ou désactive la sélection à la souris sur un canevas. Quand false (valeur par défaut), aucun rectangle de sélection ne peut être tracé. Le style reste configurable dans tous les cas.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
enabledvrai/fauxtrue pour activer, false pour désactiver (défaut: false)
Retourne: vrai/faux — true si réussi, false si le canevas n'existe pas.
voh.canvas.selection.setEnabled(canvas, true);   // activer
voh.canvas.selection.setEnabled(canvas, false);  // désactiver
voh.canvas.selection.getEnabled(canvasId) → vrai/faux ou null
Retourne l'état d'activation de la sélection à la souris.
Retourne: vrai/faux — true si activée, false si désactivée, null si le canevas n'existe pas.
console.log(voh.canvas.selection.getEnabled(canvas));  // true ou false

Bouton souris

voh.canvas.selection.setMouseButton(canvasId, button) → vrai/faux
Définit le bouton souris utilisé pour tracer le rectangle de sélection. Valeurs acceptées: 'left' (défaut), 'middle', 'right'. Un bouton invalide est ignoré. Quand le bouton droit est configuré, le menu contextuel est bloqué automatiquement.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
buttontexte'left', 'middle' ou 'right' (défaut: 'left')
Retourne: vrai/faux — true si réussi, false si le canevas n'existe pas ou bouton invalide.
voh.canvas.selection.setMouseButton(canvas, 'left');    // bouton gauche (défaut)
voh.canvas.selection.setMouseButton(canvas, 'right');   // bouton droit
voh.canvas.selection.setMouseButton(canvas, 'middle');  // molette
voh.canvas.selection.getMouseButton(canvasId) → texte ou null
Retourne le bouton souris configuré pour la sélection.
Retourne: texte — 'left', 'middle' ou 'right', ou null si le canevas n'existe pas.
console.log(voh.canvas.selection.getMouseButton(canvas));  // 'left'

Couleur de fond

voh.canvas.selection.setBackgroundColor(canvasId, color) → vrai/faux
Définit la couleur de fond du rectangle de sélection. Utilisez rgba() pour contrôler la transparence — un fond semi-transparent est recommandé pour que les objets restent visibles sous le rectangle.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
colortexteCouleur CSS — ex: 'rgba(0, 120, 215, 0.15)' (défaut)
Retourne: vrai/faux — true si réussi, false si le canevas n'existe pas.
voh.canvas.selection.setBackgroundColor(canvas, 'rgba(0, 120, 215, 0.15)');  // bleu (défaut)
voh.canvas.selection.setBackgroundColor(canvas, 'rgba(0, 200, 80, 0.10)');   // vert
voh.canvas.selection.getBackgroundColor(canvasId) → texte ou null
Retourne la couleur de fond actuelle du rectangle de sélection.
Retourne: texte — la valeur CSS, ou null si le canevas n'existe pas.
console.log(voh.canvas.selection.getBackgroundColor(canvas));  // 'rgba(0, 120, 215, 0.15)'

voh.canvas.selection.setBorderColor(canvasId, color) → vrai/faux
Définit la couleur de la bordure du rectangle de sélection. Accepte toute valeur CSS valide, dont rgba() pour la transparence.
ParamètreTypeDescription

voh.canvas.selection.setBorderColor(canvas, 'rgba(0, 180, 60, 0.9)');   // vert
voh.canvas.selection.getBorderColor(canvasId) → texte ou null

Épaisseur de bordure

voh.canvas.selection.setBorderThickness(canvasId, thickness) → vrai/faux
Définit l'épaisseur de la bordure du rectangle de sélection, en pixels. La valeur est bornée à 0 minimum. 0 = pas de bordure visible.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
thicknessnombreÉpaisseur en pixels, minimum 0 (défaut: 1)
Retourne: vrai/faux — true si réussi, false si le canevas n'existe pas.
voh.canvas.selection.setBorderThickness(canvas, 1);  // défaut
voh.canvas.selection.setBorderThickness(canvas, 2);  // plus épais
voh.canvas.selection.setBorderThickness(canvas, 0);  // pas de bordure
voh.canvas.selection.getBorderThickness(canvasId) → nombre ou null
Retourne l'épaisseur actuelle de la bordure du rectangle de sélection.
Retourne: nombre — l'épaisseur en pixels, ou null si le canevas n'existe pas.
console.log(voh.canvas.selection.getBorderThickness(canvas));  // 1

Style de bordure

voh.canvas.selection.setBorderStyle(canvasId, style) → vrai/faux
Définit le style du trait de la bordure du rectangle de sélection. Valeurs acceptées: 'solid' (défaut), 'dashed', 'dotted'. Un style invalide est ignoré.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
styletexte'solid' (défaut), 'dashed' ou 'dotted'
Retourne: vrai/faux — true si réussi, false si le canevas n'existe pas ou style invalide.
voh.canvas.selection.setBorderStyle(canvas, 'solid');   // continu (défaut)
voh.canvas.selection.setBorderStyle(canvas, 'dashed');  // tiretés
voh.canvas.selection.setBorderStyle(canvas, 'dotted');  // pointillés
voh.canvas.selection.getBorderStyle(canvasId) → texte ou null
Retourne le style de bordure actuel du rectangle de sélection.
Retourne: texte — 'solid', 'dashed' ou 'dotted', ou null si le canevas n'existe pas.
console.log(voh.canvas.selection.getBorderStyle(canvas));  // 'solid'

Définir à la création

Toutes ces propriétés peuvent être définies directement dans les paramètres de voh.canvas.create(). 
const canvas = voh.canvas.create(zone, {
    name:                    'Mon canevas',
    selectionEnabled:         true,
    selectionMouseButton:    'left',
    selectionBackgroundColor: 'rgba(0, 120, 215, 0.15)',
    selectionBorderColor:     'rgba(0, 120, 215, 0.8)',
    selectionBorderThickness: 1,
    selectionBorderStyle:     'solid'
});

Simulation programmatique

voh.canvas.selection.simulate(canvasId, x1, y1, x2, y2, options) → Promise
Simule programmatiquement un tracé de rectangle de sélection à la souris, sans interaction physique.

La simulation émet les mêmes événements qu'un vrai tracé souris (canvas:selectionStarted et canvas:selectionEnded).

Si un tracé souris est déjà en cours, il est annulé proprement avant de lancer la simulation.
Pendant la simulation, la souris est ignorée — l'utilisateur ne peut pas interférer.
Les overrides de style (backgroundColor, borderColor, etc.) sont appliqués uniquement le temps de la simulation, puis le style d'origine est restitué automatiquement.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
x1nombreX de départ du tracé (pixels, relatif au canevas)
y1nombreY de départ du tracé (pixels, relatif au canevas)
x2nombreX de fin du tracé (pixels, relatif au canevas)
y2nombreY de fin du tracé (pixels, relatif au canevas)
options.durationnombreDurée de l'animation en ms. 0 = instantané (défaut: 600)
options.backgroundColortexteOverride couleur de fond pendant la simulation (facultatif)
options.borderColortexteOverride couleur de bordure pendant la simulation (facultatif)
options.borderThicknessnombreOverride épaisseur de bordure pendant la simulation (facultatif)
options.borderStyletexteOverride style de bordure pendant la simulation (facultatif)
Retourne: Promise<boolean> — Résout true quand le tracé est terminé, false si la simulation a été annulée en cours de route.
// Simulation simple — tracé animé en 800ms avec le style courant du canevas
await voh.canvas.selection.simulate(canvas, 50, 50, 300, 200, { duration: 800 });

// Simulation avec style temporaire (vert tiretés) — restitue le style d'origine à la fin
await voh.canvas.selection.simulate(canvas, 50, 50, 300, 200, {
    duration:        1000,
    backgroundColor: 'rgba(0, 200, 80, 0.12)',
    borderColor:     'rgba(0, 200, 80, 0.9)',
    borderThickness: 2,
    borderStyle:     'dashed'
});

// Mode instantané (duration: 0) — émet les événements sans animation
await voh.canvas.selection.simulate(canvas, 50, 50, 300, 200, { duration: 0 });

// Enchaîner plusieurs simulations avec await
await voh.canvas.selection.simulate(canvas, 20, 20, 150, 100, { duration: 600 });
await voh.canvas.selection.simulate(canvas, 200, 150, 500, 400, { duration: 600 });
voh.canvas.selection.cancelSimulation(canvasId) → vrai/faux
Annule une simulation de sélection en cours sur un canevas.
Le rectangle est immédiatement masqué, le style d'origine est restitué, et la Promise de simulate() se résout avec false.
Si aucune simulation n'est en cours, retourne false sans rien faire.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
Retourne: vrai/faux — true si une simulation a été annulée, false si aucune n'était en cours ou si le canevas n'existe pas.
// Lancer une simulation longue en arrière-plan
const promesse = voh.canvas.selection.simulate(canvas, 50, 50, 400, 300, { duration: 5000 });

// L'annuler après 2 secondes
setTimeout(() => {
    voh.canvas.selection.cancelSimulation(canvas);
}, 2000);

const resultat = await promesse;  // false — annulée
console.log(resultat);  // false
Événements: ces méthodes émettent des événements canvas capturables via voh.on() — canvas:selectionStarted, canvas:selectionEnded, canvas:selectionEnabledChanged, etc.
→ Voir la documentation complète dans Événements canvas.
