Événements — Page

Les 15 types d'événements émis par les pages de canevas. Chaque modification émet un événement capté par voh.on().

Liste des événements

ÉvénementDonnées reçuesQuand
page:addedcanvasId, pageId, nameUne page est ajoutée
page:removedcanvasId, pageId, nameUne page est supprimée
page:activatedcanvasId, pageId, previousPageIdLa page active change
page:renamedcanvasId, pageId, name, previousNameLe nom de la page change
page:movedcanvasId, pageId, indexL'ordre de la page change
page:duplicatedcanvasId, sourcePageId, newPageIdUne page est dupliquée
page:resizedcanvasId, pageId, property, valueLes dimensions changent
page:backgroundChangedcanvasId, pageId, property, valueLe fond change
page:borderChangedcanvasId, pageId, property, valueLa bordure change
page:gridChangedcanvasId, pageId, property, valueLa grille change
page:cursorChangedcanvasId, pageId, cursorLe curseur change
page:selectionStyleChangedcanvasId, pageId, property, valueLe style de sélection change
page:undonecanvasId, pageId, actionUne action est annulée
page:redonecanvasId, pageId, actionUne action est refaite
page:historyClearedcanvasId, pageIdL'historique est vidé
Note: Les événements ne sont émis que si la valeur change réellement (pas de doublon).

page:added

Émis quand une page est ajoutée au canevas avec voh.canvas.pages.add().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la nouvelle page
nametexteLe nom de la page
voh.on('page:added', (data) => {
    console.log('Nouvelle page:', data.pageId, data.name);
});

page:removed

Émis quand une page est supprimée avec voh.canvas.pages.remove(). Le nom est capturé avant la suppression pour être disponible dans l'événement.
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page supprimée
nametexteLe nom de la page (avant suppression)
voh.on('page:removed', (data) => {
    console.log('Page supprimée:', data.name);
});

page:activated

Émis quand la page active change avec setActive(), next(), previous() ou goTo(). Également émis automatiquement lors de l'ajout de la première page ou après la suppression de la page active.
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la nouvelle page active
previousPageIdnombre ou nullL'ID de la page précédemment active. null si première activation
voh.on('page:activated', (data) => {
    console.log('Page active:', data.pageId);
    console.log('Précédente:', data.previousPageId);
});

page:renamed

Émis quand le nom d'une page est modifié avec voh.canvas.pages.setName().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
nametexteLe nouveau nom
previousNametexteL'ancien nom
voh.on('page:renamed', (data) => {
    console.log(data.previousName, '→', data.name);
});

page:moved

Émis quand l'ordre d'une page change avec moveUp(), moveDown() ou moveTo().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page déplacée
indexnombreNouvel index (position) dans la liste
voh.on('page:moved', (data) => {
    console.log('Page', data.pageId, '→ index', data.index);
});

page:duplicated

Émis quand une page est dupliquée avec voh.canvas.pages.duplicate().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
sourcePageIdnombreL'ID de la page source
newPageIdnombreL'ID de la nouvelle page créée
voh.on('page:duplicated', (data) => {
    console.log('Page', data.sourcePageId, '→ copie', data.newPageId);
});

page:resized

Émis quand les dimensions d'une page changent avec setWidth() ou setHeight().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
propertytexte'width' ou 'height'
valuenombreLa nouvelle valeur en pixels
voh.on('page:resized', (data) => {
    console.log(data.property, '=', data.value);
    // "width" = 1024
});

page:backgroundChanged

Émis quand le fond d'une page change: couleur, image ou opacité de l'image.
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
propertytexte'color', 'image' ou 'imageOpacity'
valuevariableLa nouvelle valeur
voh.on('page:backgroundChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "color" = "#1a1a2e"
});

page:borderChanged

Émis quand la bordure d'une page change: couleur, épaisseur, style, arrondi ou visibilité.
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
propertytexteLa propriété modifiée (ex: 'borderColor', 'borderThickness')
valuevariableLa nouvelle valeur
voh.on('page:borderChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "borderColor" = "rgba(0, 0, 0, 0.5)"
});

page:gridChanged

Émis quand la grille d'une page change: taille de cellule, couleur, opacité, épaisseur, style, visibilité ou snap.
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
propertytexteLa propriété modifiée (ex: 'gridCellWidth', 'gridIsVisible')
valuevariableLa nouvelle valeur
voh.on('page:gridChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "gridCellWidth" = 50
    // "gridIsVisible" = true
});

page:cursorChanged

Émis quand le curseur d'une page change avec voh.canvas.pages.cursor.set().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
cursortexteLe nouveau curseur CSS (ex: 'crosshair')
voh.on('page:cursorChanged', (data) => {
    console.log('Curseur:', data.cursor);
});

page:selectionStyleChanged

Émis quand le style de sélection souris d'une page change: activation, bouton, couleurs ou bordure.
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
propertytexteLa propriété modifiée (ex: 'selectionEnabled', 'selectionBorderColor')
valuevariableLa nouvelle valeur
voh.on('page:selectionStyleChanged', (data) => {
    console.log(data.property, '=', data.value);
    // "selectionEnabled" = true
});

page:undone

Émis quand une action est annulée sur cette page avec voh.canvas.pages.history.undo().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
actiontexteDescription de l'action annulée
voh.on('page:undone', (data) => {
    console.log('Annulé:', data.action);
    // "Annulé: backgroundColor: #fff → #1a1a2e"
});

page:redone

Émis quand une action est refaite sur cette page avec voh.canvas.pages.history.redo().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
actiontexteDescription de l'action refaite
voh.on('page:redone', (data) => {
    console.log('Refait:', data.action);
});

page:historyCleared

Émis quand l'historique undo/redo d'une page est vidé avec voh.canvas.pages.history.clear().
PropriétéTypeDescription
canvasIdnombreL'ID du canevas
pageIdnombreL'ID de la page
voh.on('page:historyCleared', (data) => {
    console.log('Historique vidé pour page', data.pageId);
});
