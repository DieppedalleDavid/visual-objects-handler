voh.canvas.history — Undo/Redo

Raccourci pour annuler et refaire les actions sur la page active d'un canevas. Toutes les modifications (propriétés de page, création/suppression/modification d'objets) sont automatiquement enregistrées.

Ces méthodes agissent sur la page active du canevas. Pour cibler une page spécifique, utiliser voh.canvas.pages.history.* avec le paramètre pageId.

Annuler / Refaire

voh.canvas.history.undo(canvasId) → vrai/faux
Annule la dernière action sur la page active. Le rendu visuel est automatiquement mis à jour.
ParamètreTypeDescription
canvasIdnombreID du canevas
Retourne: vrai/faux — true si annulée, false si rien à annuler.
voh.canvas.history.undo(canvas);  // annuler la dernière modification
voh.canvas.history.redo(canvasId) → vrai/faux
Refait la dernière action annulée. Le rendu visuel est automatiquement mis à jour.
ParamètreTypeDescription
canvasIdnombreID du canevas
Retourne: vrai/faux — true si refaite, false si rien à refaire.
voh.canvas.history.redo(canvas);  // refaire
voh.canvas.history.canUndo(canvasId) → vrai/faux
Retourne true s'il y a au moins une action annulable.
Retourne: vrai/faux
if (voh.canvas.history.canUndo(canvas)) {
    voh.canvas.history.undo(canvas);
}
voh.canvas.history.canRedo(canvasId) → vrai/faux
Retourne true s'il y a au moins une action refaisable.
Retourne: vrai/faux
console.log('Peut refaire:', voh.canvas.history.canRedo(canvas));

Compteurs

voh.canvas.history.getUndoCount(canvasId) → nombre
Retourne le nombre d'actions annulables.
Retourne: nombre
console.log('Undo:', voh.canvas.history.getUndoCount(canvas));  // ex: 5
voh.canvas.history.getRedoCount(canvasId) → nombre
Retourne le nombre d'actions refaisables.
Retourne: nombre
console.log('Redo:', voh.canvas.history.getRedoCount(canvas));  // ex: 2

Configuration

voh.canvas.history.setMaxLevel(canvasId, maxLevel) → vrai/faux
Définit le nombre maximum d'actions conservées dans la pile undo (défaut: 50, minimum: 1). Si la pile dépasse le nouveau max, les plus anciennes sont éjectées.
ParamètreTypeDescription
canvasIdnombreID du canevas
maxLevelnombreNombre maximum d'actions (minimum 1)
Retourne: vrai/faux — true si réussi.
voh.canvas.history.setMaxLevel(canvas, 100);  // garder 100 actions
voh.canvas.history.getMaxLevel(canvasId) → nombre | null
Retourne le niveau maximum de l'historique.
Retourne: nombre | null — le max, ou null si erreur.
console.log(voh.canvas.history.getMaxLevel(canvas));  // 100
voh.canvas.history.clear(canvasId) → vrai/faux
Vide entièrement l'historique undo et redo de la page active.
Retourne: vrai/faux — true si réussi.
voh.canvas.history.clear(canvas);  // vider tout l'historique

Batch (pause / resume)

voh.canvas.history.pause(canvasId) → vrai/faux
Met l'historique en pause. Toutes les modifications effectuées pendant la pause seront regroupées en une seule action à l'appel de resume(). Les pauses sont imbriquables (compteur).
ParamètreTypeDescription
canvasIdnombreID du canevas
Retourne: vrai/faux — true si réussi.
// 3 modifications = 1 seul undo
voh.canvas.history.pause(canvas);
voh.canvas.pages.background.setColor(canvas, 'rgba(50, 50, 60, 1.0)');
voh.canvas.pages.border.setVisible(canvas, true);
voh.canvas.pages.border.setColor(canvas, 'rgba(255, 100, 0, 0.8)');
voh.canvas.history.resume(canvas, 'Style sombre + bordure orange');

// Un seul undo annule les 3 modifications
voh.canvas.history.undo(canvas);
voh.canvas.history.resume(canvasId, description?) → vrai/faux
Reprend l'enregistrement. Les actions accumulées pendant la pause deviennent une seule action batch. La description optionnelle nomme le batch.
ParamètreTypeDescription
canvasIdnombreID du canevas
descriptiontexteDescription du batch (optionnel)
Retourne: vrai/faux — true si réussi, false si pas en pause.
voh.canvas.history.resume(canvas, 'Changement de thème');
Événements: les opérations émettent page:undone, page:redone, page:historyCleared.
Voir aussi: voh.canvas.pages.history.* pour cibler une page spécifique avec pageId.
