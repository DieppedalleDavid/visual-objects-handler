voh.canvas — Position

Chaque canvas a une position X, Y dans sa zone. La position représente le coin haut-gauche du canvas.

voh.canvas.setX(canvasId, x) → vrai/faux
Définit la position horizontale du canvas dans sa zone.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
xnombrePosition X en pixels
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setX(canvas1, 100);
voh.canvas.getX(canvasId) → nombre ou null
Retourne la position X du canvas.
Retourne: nombre — la position X en pixels, ou null si le canvas n'existe pas.
console.log(voh.canvas.getX(canvas1));  // 100
voh.canvas.setY(canvasId, y) → vrai/faux
Définit la position verticale du canvas dans sa zone.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
ynombrePosition Y en pixels
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setY(canvas1, 80);
voh.canvas.getY(canvasId) → nombre ou null
Retourne la position Y du canvas.
Retourne: nombre — la position Y en pixels, ou null si le canvas n'existe pas.
console.log(voh.canvas.getY(canvas1));  // 80
