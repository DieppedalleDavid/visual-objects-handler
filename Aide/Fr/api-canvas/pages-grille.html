voh.canvas.pages.grid — Grille / Snap par page

Chaque page a sa propre grille avec son propre accrochage (snap). La grille est rendue par Pixi.js (GPU) avec 4 styles: solid, dashed, dotted, dots.

Règle fondamentale: sans pageId = page active du canvas. Avec pageId = page ciblée. La Page 1 est créée automatiquement avec chaque canvas. Il doit toujours rester au moins une page.

Visibilité

voh.canvas.pages.grid.setVisible(canvasId, visible, pageId?) → vrai/faux
Affiche ou masque la grille (défaut: false).
ParamètreTypeDescription
canvasIdnombreID du canvas
visiblevrai/fauxLa valeur à définir
pageIdnombreID de la page (optionnel, défaut: page active)
Retourne: vrai/faux — true si réussi, false si canvas/page inexistant.
voh.canvas.pages.grid.setVisible(canvas, true);
voh.canvas.pages.grid.getVisible(canvasId, pageId?) → vrai/faux | null
Retourne la visibilité.
ParamètreTypeDescription
canvasIdnombreID du canvas
pageIdnombreID de la page (optionnel)
Retourne: vrai/faux | null — la valeur, ou null si canvas/page inexistant.
console.log(voh.canvas.pages.grid.getVisible(canvas));
