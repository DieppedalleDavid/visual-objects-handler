voh.canvas — Dimensions

Largeur et hauteur d'un canvas en pixels.

voh.canvas.setWidth(canvasId, width) → vrai/faux
Définit la largeur du canvas.
ParamètreTypeDescription
canvasIdnombreL'ID du canvas
widthnombreLargeur en pixels
Retourne: vrai/faux — true si réussi, false si canvas inexistant.
voh.canvas.setWidth(canvas1, 700);
voh.canvas.getWidth(canvasId) → nombre ou null
Retourne la largeur du canvas.
Retourne: nombre — la largeur en pixels, ou null si le canvas n'existe pas.
console.log(voh.canvas.getWidth(canvas1));  // 700
voh.canvas.setHeight(canvasId, height) → vrai/faux
