voh.canvas — Fond et rendu

Couleur de fond du canevas et contrôle du moteur de rendu Pixi.js.

Couleur de fond

voh.canvas.setBackgroundColor(canvasId, color) → vrai/faux
Définit la couleur de fond du canevas sur la page active. Accepte toute couleur CSS valide (hex, rgb, rgba, nom). La couleur est stockée dans les données de la page active.
ParamètreTypeDescription
canvasIdnombreL'ID du canevas
colortexteCouleur CSS (ex: '#ff0000', 'rgba(0,0,0,0.5)')
Retourne: vrai/faux
voh.canvas.setBackgroundColor(canvas, '#1a1a2e');
voh.canvas.getBackgroundColor(canvasId) → texte | null
Retourne la couleur de fond de la page active, ou null si le canevas n'existe pas.
Retourne: texte | null
console.log(voh.canvas.getBackgroundColor(canvas));  // 'rgba(255, 255, 255, 1.0)'

Attendre le rendu

voh.canvas.waitForRender(canvasId) → Promise<vrai>
Attend que le rendu Pixi.js soit affiché à l'écran. Si le renderer n'est pas encore initialisé (init asynchrone en cours), attend qu'il le soit. Force un rendu puis attend deux frames du navigateur pour garantir que les pixels sont visibles. Utile après la création d'un canevas ou d'objets.
Retourne: Promise<vrai> — Résout true quand l'affichage est à jour.
await voh.canvas.waitForRender(canvas);
// ici les pixels sont visibles, on peut continuer

Statistiques de rendu

voh.canvas.getRenderStats(canvasId) → objet | null
Retourne les statistiques de rendu complètes du canevas.
Retourne: { lastTime, minTime, maxTime, averageTime, count, totalTime } — Temps en millisecondes.
const stats = voh.canvas.getRenderStats(canvas);
console.log('Rendus:', stats.count, 'Moyenne:', stats.averageTime.toFixed(2), 'ms');
voh.canvas.getLastRenderTime(canvasId) → nombre
Retourne le temps du dernier rendu en millisecondes.
Retourne: nombre — Temps en ms (ex: 0.342).
console.log(voh.canvas.getLastRenderTime(canvas));  // 0.342
voh.canvas.resetRenderStats(canvasId)
Remet à zéro les statistiques de rendu du canevas (compteur, min, max, total).
voh.canvas.resetRenderStats(canvas);
