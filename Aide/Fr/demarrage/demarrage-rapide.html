Démarrage rapide

Pour utiliser VOH, il faut 3 étapes : inclure les fichiers, créer une instance, et initialiser.

1. Inclure Pixi.js et les fichiers Core dans votre page HTML

<!-- Moteur de rendu GPU (obligatoire) -->
<script src="Libraries/Pixi/Pixi.js"></script>

<!-- Core VOH (dans cet ordre !) -->
<script src="Core/Core-Diagnostics.js"></script>
<script src="Core/Core-EventEmitter.js"></script>
<script src="Core/Core-API.js"></script>
<script src="Core/Core-Zone.js"></script>
<script src="Core/Core-CanvasSelection.js"></script>
<script src="Core/Core-CanvasHistory.js"></script>
<script src="Core/Core-CanvasPage.js"></script>
<script src="Core/Core-RenderEngine.js"></script>
<script src="Core/Core-Canvas.js"></script>
<script src="Core/Core-Objects.js"></script>
<script src="Core/Core-ObjectSelection.js"></script>
<script src="Core/Core-SpatialGrid.js"></script>
<script src="Core/Core-ObjectInteraction.js"></script>
<script src="Core/Core-Main.js"></script>

2. Créer une instance et l'initialiser

// Créer VOH dans un <div> de votre page
const voh = new VisualObjectsHandler(document.getElementById('container'));

// Initialiser le moteur (obligatoire, à faire une seule fois)
await voh.init();

3. Créer une zone et commencer à travailler

// Créer une zone avec un fond jaune et une bordure rouge
const zone = voh.zone.create({
    name: 'Ma première zone',
    backgroundColor: 'rgba(255, 200, 0, 1.0)',
    borderColor: 'rgba(255, 0, 0, 1.0)',
    borderThickness: 2
});

// zone contient l'ID (un nombre) de la zone créée
console.log('Zone créée, ID:', zone);
Deux lignes obligatoires au début de chaque script :
const voh = new VisualObjectsHandler(document.getElementById('container')); crée le moteur.
await voh.init(); l'initialise. Sans ça, rien ne fonctionne.

Méthodes globales

voh.init() → Promise
Initialise le moteur VOH. Doit être appelé une seule fois, avant toute autre opération. Retourne une Promise — utiliser await pour attendre la fin de l'initialisation. 
const voh = new VisualObjectsHandler(document.getElementById('container'));
await voh.init();
// Le moteur est prêt, on peut créer des zones et des canevas
voh.isReady() → vrai/faux
Retourne true si voh.init() a été appelé et s'est terminé avec succès. Utile pour vérifier l'état du moteur avant de l'utiliser. 
if (voh.isReady()) {
    console.log('Moteur prêt');
} else {
    console.log('Moteur non initialisé');
}
voh.getVersion() → texte
Retourne la version du Core VOH sous forme de chaîne (ex: '2.26.00'). Utile pour le débogage ou pour vérifier la compatibilité. 
console.log(voh.getVersion());  // ex: '2.26.00'
voh.destroy() → rien
Détruit complètement l'instance VOH : libère les ressources Pixi.js, supprime les listeners DOM, vide toutes les zones, canevas et objets. À appeler quand le moteur n'est plus nécessaire (navigation SPA, fermeture d'une fenêtre, nettoyage de tests). Après un destroy(), l'instance ne peut plus être utilisée. 
// Nettoyer le moteur avant de quitter
voh.destroy();
